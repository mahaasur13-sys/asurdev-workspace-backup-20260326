## ЧАСТЬ 1: ХОРАРНАЯ АСТРОЛОГИЯ (HORARY)

### Фундаментальные принципы Фроули

**Что такое хорарная астрология:**
Хорар (от лат. "hora" = час) — отвечает на конкретный вопрос, заданный в момент, когда астролог "принимает" вопрос.

**Ключевые правила Фроули:**

1. **Вопрос должен быть конкретным** — "Будет ли X?" не хорар, а "Когда X?" — хорар
2. **Время фиксируется в момент получения** — это начало суждения
3. **Significators** — планеты-управители для Querent (вопрошающий) и Quesited (предмет вопроса)
4. **Критерий радикальности** — планеты в углах (ASC/DESC/MC/IC) = сильный вопрос

**Zettelkasten теги хорара:**
`#horary #question #querent #quesited #significator #angular #cadent #succedent #radical #void-of-course #Moon #aspecting #applying #separating #dignity #debility #accidental-dignity #essential-dignity # reception #translation #collection #witnessing`

from rag.embedder import AstroEmbedder, ChromaStore
embedder = AstroEmbedder()
store = ChromaStore(persist_directory='./data/rag_index', collection_name='asurdev')

# Index Frawley's Horary + Electional
frawley_text = open('docs/horary-electional-astrology-frawley-zettelkasten.md').read()
chunks = embedder.chunk_text(frawley_text, chunk_size=500, overlap=50)
for i, chunk in enumerate(chunks):
    store.add_documents([chunk], [{'source': 'frawley-horary-electional', 'chunk': i}])
print(f'Indexed {len(chunks)} Frawley chunks')
