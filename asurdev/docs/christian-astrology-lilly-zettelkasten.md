**Чанк 2:** House-by-house breakdown — Vita, Pecunia, Locum, Genitor, Nati, Valetudo, Uxor, Mors, Pietas, Munus, Lucrum, Carcer

**Добавить в RAG:**
```bash
cd /home/workspace/asurdevSentinel && python3 -c "
from rag.embedder import AstroEmbedder, ChromaStore
embedder = AstroEmbedder()
store = ChromaStore(persist_directory='./data/rag_index', collection_name='asurdev')
# Index Lilly's Christian Astrology
lilly_text = open('docs/christian-astrology-lilly-zettelkasten.md').read()
chunks = embedder.chunk_text(lilly_text, chunk_size=500, overlap=50)
for i, chunk in enumerate(chunks):
    store.add_documents([chunk], [{'source': 'lilly-christian-astrology', 'chunk': i}])
print(f'Indexed {len(chunks)} Lilly chunks')
"
```
