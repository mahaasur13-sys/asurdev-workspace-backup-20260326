# Smart Money Concepts (SMC) Agent

## Обзор

**SMC Agent** — агент институционального анализа, отслеживающий действия крупных игроков (банков, хедж-фондов).

## Стратегии

### 1. Order Blocks (Ордерные блоки)
Последняя свеча перед сильным импульсным движением — зона набора позиции институционалами.

```
Bullish OB: свеча перед сильным ростом → поддержка для входа в лонг
Bearish OB: свеча перед сильным падением → сопротивление для входа в шорт
```

### 2. Fair Value Gaps (FVG)
Пробелы на графике — зоны несбалансированного спроса/предложения. Цена возвращается заполнить пробел.

### 3. Liquidity Sweeps (Захват ликвидности)
Институционалы "выносят" стоп-лоссы розничных трейдеров:
- Свип выше максимумов → ловушка для шортеров
- Свип ниже минимумов → ловушка для лонгистов

### 4. Market Phase Detection
- **Accumulation** — накопление (боковик, низкая волатильность)
- **Distribution** — распределение (сильное движение вверх)
- **Manipulation** — манипуляция (смешанные сигналы)

## Сигналы

| Тип | Направление | Entry | Stop Loss | Risk/Reward |
|-----|-------------|-------|-----------|-------------|
| Order Block | Bull/Bear | Зона OB | За экстремум | 1:2 |
| Liquidity Sweep | Отскок | После свипа | За уровень | 1:1.5 |
| FVG | По тренду | К зоне FVG | За гэп | 1:2 |

## Практический чек-лист

```
1. Определи фазу рынка
2. Найди ключевые уровни (свинг хай/лоу)
3. Найди ордерные блоки и FVG
4. Дождись захвата ликвидности
5. Войди после подтверждения
6. Управляй риском (0.5-1% на сделку)
```

## Использование

```python
from smc import get_smc_agent, Candle

agent = get_smc_agent()
result = agent.analyze(candles, "BTCUSDT")

print(result['market_phase'])  # Текущая фаза
print(result['signals'])       # Торговые сигналы
print(result['summary'])       # Резюме
```
