## 🎯 Что это такое?

**asurdev Sentinel** — это команда из 7 "умных помощников" (агентов), которые смотрят на график криптовалюты и говорят тебе:

> "Купи", "Продай" или "Подожди"

Каждый помощник думает по-своему:
- Один смотрит на **цену и объёмы**
- Другой изучает **математику Ганна**
- Третий **астрологию** считает
- И так далее...

Потом они "голосуют" и дают тебе **финальную рекомендацию**.

## 📊 Агенты — кто они?

### 1️⃣ Market Analyst (Аналитик рынка)
**Что делает:** Смотрит на график и ищет тренды.

```
График: 📈📈📈📉📉📉

Смотрит:
- Цена выше или ниже средней линии (MA)?
- RSI показывает перекупленность/перепроданность?
- Объёмы подтверждают движение?
```

**Результат:** "Цена растёт, RSI в норме — бычий сигнал (75%)"

---

### 2️⃣ Bull Researcher (Бычий исследователь)
**Что делает:** Ищет ВСЕ причины почему цена может вырасти.

```
Ищет:
- Позитивные новости
- Партнёрства и релизы
- Рост активности адресов
- Whale накопление
```

**Результат:** "Нашёл 5 бычьих факторов: ETF inflow, whale buying..."

---

### 3️⃣ Bear Researcher (Медвежий исследователь)
**Что делает:** Ищет ВСЕ причины почему цена может упасть.

```
Ищет:
- Негативные новости
- Регуляторные риски
- Продажи китов
- Макро давление
```

**Результат:** "Нашёл 3 медвежьих фактора: Profit-taking, high fees..."

---

### 4️⃣ SMC Agent (Smart Money Concepts)
**Что делает:** Находит где "умные деньги" покупают/продают.

```
Концепция:
- Order Blocks = зоны где институционалы покупали
- Fair Value Gaps = "дыры" на графике которые нужно заполнить
- Liquidity Pools = где "выносят" стопы
```

**Результат:** "ETH в зоне bull OB, вероятен рост к $2000"

---

### 5️⃣ Monte Carlo Agent (Монте-Карло)
**Что делает:** Запускает 10,000 симуляций "а что если".

```
Симуляция:
Бросает монетку 10,000 раз с верами:
- Если тренд вверх — чаще "орел"
- Если высокая волатильность — разброс больше

Смотрит:
- В 5% случаев цена была бы...
- В 50% случаев цена была бы...
- В 95% случаев цена была бы...
```

**Результат:** "Через 30 дней: пессимизм $42,000, медиана $55,000, оптимизм $68,000"

---

### 6️⃣ Gann Agent (Ганн)
**Что делает:** Использует математику Ганна (Square of 9).

```
Ганн верил:
- Цена и время связаны
- Квадратные числа важны
- Углы 45° показывают тренд

Инструменты:
- Square of 9 — ценовые уровни
- Death Zones — даты когда рынок "умирает"
- Gann Angles — линии тренда
```

**Результат:** "Ближайшая Death Zone: 15-20 марта, не входить в лонг"

---

### 7️⃣ Dow Theory Agent (Теория Доу)
**Что делает:** Определяет тренд по классике 1900 года.

```
Правила Доу:
- Higher High + Higher Low = Восходящий тренд 📈
- Lower High + Lower Low = Нисходящий тренд 📉
- HH + LH = Возможный разворот ↔️

Нужны:
- Bitcoin
- Ethereum
- Общий рынок (чтобы совпадали!)
```

**Результат:** "Восходящий тренд подтверждён на BTC, ETH, Total Cap"

## 🗳️ Synthesizer — как голосуем

Каждый агент говорит: "BUY" или "SELL" или "HOLD" + процент уверенности.

```
Голосование:

Market Analyst:  BUY (75%)
Bull Researcher: BUY (80%)
Bear Researcher: SELL (60%)
SMC Agent:       BUY (70%)
Monte Carlo:     HOLD (50%)
Gann Agent:      SELL (65%)
Dow Theory:      BUY (72%)
                        ↓
Подсчёт: BUY=4 голоса(75%), SELL=2 голоса(62%), HOLD=1 голос(50%)
                        ↓
Взвешивание по уверенности:
BUY сила = 75+80+70+72 = 297
SELL сила = 60+65 = 125
                        ↓
ИТОГ: BUY (297 vs 125) — 70% уверенность
```

## 📈 Как данные проходят через систему

### Шаг 1: Получаем данные
```
Coingecko API → BTC price, volumes, OHLC
```

### Шаг 2: Параллельный анализ
```
ВСЕ 7 агентов анализируют одновременно
(как 7 экспертов смотрят на один график)
```

### Шаг 3: Сбор результатов
```
Market: "BUY 75%"
Bull: "BUY 80%"
...
```

### Шаг 4: Синтез
```
Synthesizer считает голоса
→ Взвешивает по уверенности
→ Формирует C.L.E.A.R. отчёт
```

### Шаг 5: Результат
```
🟢 BUY (70%)

Context: Цена пробила MA200, объёмы растут
Logic: Быки доминируют, RSI в норме
Evidence: 
  Bulls: 5 факторов роста
  Bears: 2 фактора риска
  Astro: Благоприятный день для покупок
Assessment: Risk 3/10
Recommendation: BUY зона $48,000-49,000, SL $45,000

⚠️ Это советник-компаньон. Решение за тобой.
```

## 💾 Память и обучение

### Как работает память?

```
Каждый запрос получает UNIQUE ID

Request #12345
├── Market Analyst: BUY 75%
├── Bull: BUY 80%
├── ...
└── Final: BUY 70%
    │
    └── Пользователь нажал 👍 (agree)
        │
        └── Это записывается в Quality DB
            │
            └── Через 100 запросов:
                - Market Analyst правильный в 68% случаев
                - → Его вес повышается
```

### RAG (Retrieval Augmented Generation)

```
Ты спрашиваешь: "Что по Elliott Waves?"

Система:
1. Ищет в базе похожие запросы
2. Находит: "Волны Эллиотта" из твоих заметок
3. Добавляет к промпту агента
4. Агент отвечает с учётом ТВОЕГО контекста
```

## 🛡️ Безопасность

### Защита от галлюцинаций:
```
1. Всегда показываем источники
2. Ограничиваем "фантазию" агентов
3. Требуем подтверждения от нескольких агентов
```

### Защита от неправильных решений:
```
1. Risk score всегда показывается
2. Stop loss всегда рекомендуется
3. Disclaimer: "Это советник, не авто-трейдер"
```

## 🚀 Быстрый старт

### 1. Установка
```bash
cd asurdevSentinel
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Запуск
```bash
# REST API
uvicorn api.main:app --reload

# Или Streamlit UI
streamlit run ui/dashboard.py
```

### 3. Использование
```bash
# Через API
curl -X POST http://localhost:8000/analyze \
  -H "Content-Type: application/json" \
  -d '{"symbol": "BTC"}'

# Через браузер
# Открыть http://localhost:8501
```
