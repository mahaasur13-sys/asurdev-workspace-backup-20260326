# Западная Мухурта — Timing for Actions

Теги: #мухурта #западная #timing #electional #翟

## Overview

Западная мухурта (Electional Astrology) использует те же принципы, что и индийская, но с акцентом на планетные влияния и аспекты.

## Ключевые факторы

### 1. Планетарные влияния

| Планета | Благоприятно для | Неблагоприятно для |
|---------|-----------------|-------------------|
| Солнце | Leadership, авторитет | - |
| Луна | Эмоции, интуиция, торговля | Конфликты |
| Меркурий | Коммуникация, торговля | - |
| Венера | Любовь, финансы, искусство | - |
| Марс | Действие, спорт | Мирные дела |
| Юпитер | Рост, удача, образование | - |
| Сатурн | Дисциплина, строительство | Начинания |

### 2. День недели

| День | Планета | Лучшее для |
|------|---------|-----------|
| Воскресенье | Солнце | Leadership, авторитет |
| Понедельник | Луна | Торговля, эмоциональные дела |
| Вторник | Марс | Действие, конфронтация |
| Среда | Меркурий | Коммуникация, контракты |
| Четверг | Юпитер | Образование, путешествия |
| Пятница | Венера | Любовь, финансы |
| Суббота | Сатурн | Строительство, карма |

### 3. Лунные фазы

| Фаза | Благоприятно для | Неблагоприятно для |
|------|-----------------|-------------------|
| New Moon | Новые начинания | Продажи |
| Waxing | Накопление, рост | Уменьшение |
| Full Moon | Завершение, публичные дела | Новые начинания |
| Waning | Избавление, реорганизация | Контракты |

### 4. Аспекты для Timing

- **Трин** — гармоничное время для начинаний
- **Секстиль** — возможности, новые проекты
- **Квадрат** — напряжение, требует усилий
- **Соединение** — слияние энергий

## Правила выбора времени

### Общее правило

1. Избегать ретроградных планет
2. Искать благоприятные аспекты к планете-управителю дела
3. Учитывать положение Луны (приоритет)
4. Проверять аспекты к ASC и MC

### Пример: Выбор времени для бизнеса

1. Юпитер в хорошем аспекте (+)
2. Луна не в квадрате к Сатурну
3. Меркурий не ретроградный
4. Избегать вторника (Марс) для мирных переговоров
