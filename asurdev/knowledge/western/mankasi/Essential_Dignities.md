# Essential Dignities — Essential Strength

Теги: #манкаси #сила #дигнити #обитель #экзальтация #падение #изгнание

## Overview

Essential Dignities — оценка силы планеты на основе её положения в знаке. Это фундаментальная концепция для определения power level планеты.

## 5 Types of Essential Dignities

### 1. Domiciliation (Обитель)

Планета в своём знаке — максимальный комфорт и сила.

| Планета | Знак(и) |
|---------|---------|
| Солнце | Лев |
| Луна | Рак |
| Меркурий | Близнецы, Дева |
| Венера | Телец, Весы |
| Марс | Овен, Скорпион |
| Юпитер | Стрелец, Рыбы |
| Сатурн | Козерог, Водолей |

### 2. Exaltation (Экзальтация)

Планета в знаке максимальной силы — даже сильнее обители.

| Планета | Знак | Градус |
|---------|------|--------|
| Солнце | Овен | 19° |
| Луна | Телец | 3° |
| Меркурий | Дева | 15° |
| Венера | Рыбы | 27° |
| Марс | Козерог | 28° |
| Юпитер | Рак | 15° |
| Сатурн | Весы | 21° |

### 3. Fall (Падение)

Противоположность экзальтации — планета ослаблена.

| Планета | Знак |
|---------|------|
| Солнце | Весы |
| Луна | Скорпион |
| Меркурий | Рыбы |
| Венера | Дева |
| Марс | Рак |
| Юпитер | Козерог |
| Сатурн | Овен |

### 4. Exile/Detriment (Изгнание)

Противоположность обители — планета максимально ослаблена.

| Планета | Знак(и) |
|---------|---------|
| Солнце | Водолей |
| Луна | Козерог |
| Меркурий | Стрелец, Рыбы |
| Венера | Овен, Скорпион |
| Марс | Весы, Телец |
| Юпитер | Близнецы, Дева |
| Сатурн | Рак, Лев |

### 5. Peregrine (Странник)

Планета не в обители, не в экзальтации, не в падении, не в изгнании. Средняя сила.

## Additional Rules

1. **Ruling Planet Dispositor** — если диспозитор силён, это усиливает планету
2. **Angular/Succedent/Falling** — положение в доме влияет на силу
3. **Conjunction with Benefics** — соединение с бенефиками усиливает
4. **Conjunction with Malefics** — соединение с малефиками ослабляет

## Использование в анализе дома

При оценке дома по Манкаси:

```
Управитель дома → Essential Dignity → Score
                → Положение в доме → Modifier
                → Кузнец → Additional Score
                → Планеты в доме → Sum
```
