# Сравнение домных систем

Теги: #манкаси #дома #система_домов #Placidus #Whole_Sign #Koch #Equal

## Overview

Разные астрологи используют разные системы для определения границ домов. Выбор системы влияет на интерпретацию.

## Основные домные системы

### 1. Whole Sign Houses (Whole Signs)

**Принцип:** Каждый знак = один дом. Асцендент определяет только знак на cusp 1 дома.

| Преимущества | Недостатки |
|-------------|------------|
| Простота | Может быть неточным для некоторых широт |
| Историческая точность | Дома могут быть очень неравными |
| Хорошо для мухурты | |

### 2. Placidus

**Принцип:** Дома делят дугу между ASC и MC на равные части по времени.

| Преимущества | Недостатки |
|-------------|------------|
| Популярность | Проблемы в высоких широтах |
| Детальные интерпретации | Перехваченные знаки |
| Хорошо для индивидуальных карт | |

### 3. Koch (Birthplace)

**Принцип:** Дома основаны на времени прохождения через меридиан.

| Преимущества | Недостатки |
|-------------|------------|
| Точность для рождения | Сложнее рассчитать |
| Хорош для хорарных карт | Менее популярен |

### 4. Equal Houses

**Принцип:** ASC = 1°, каждый дом = 30°.

| Преимущества | Недостатки |
|-------------|------------|
| Простота | MC не совпадает с 10 домом |
| Сбалансированность | |

## Когда использовать какую систему

| Ситуация | Рекомендуемая система |
|----------|---------------------|
| Мухурта | Whole Sign |
| Хорарная астрология | Koch, Placidus |
| Натальная карта | Placidus, Whole Sign |
| Высокие широты (>66°) | Whole Sign, Campanus |

## Перехваченные знаки по системам

| Система | Перехваченные знаки |
|---------|---------------------|
| Placidus | Часто Taurus/Scorpio, Leo/Aquarius |
| Whole Sign | Никогда (каждый знак = дом) |
| Koch | Реже |
| Equal | Редко |
