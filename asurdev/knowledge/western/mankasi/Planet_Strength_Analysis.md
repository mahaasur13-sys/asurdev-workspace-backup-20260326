# Planet Strength Analysis — Полный анализ силы планеты

Теги: #манкаси #планета #сила #анализ #的综合

## Overview

Комплексная оценка силы планеты с учётом всех факторов.

## Формула Total Strength

```
Total Strength = Essential Dignity Score
               + Accidental Dignity Score
               + House Position Modifier
               + Dispositor Bonus
               + Aspect Bonus/Malus
```

## Interpretation

| Total Score | Category | Interpretation |
|-------------|----------|----------------|
| 15+ | Very Strong | Планета может легко проявить свой потенциал |
| 10-14 | Strong | Хорошая способность к проявлению |
| 5-9 | Moderate | Средняя способность |
| 0-4 | Weak | Ограниченная способность |
| <0 | Very Weak | Критически ослаблена |

## Пример: Анализ Луны в натальной карте

**Входные данные:**
- Луна в Тельце, 15°
- Луна в 4 доме (Angular)
- Луна не соединена с Солнцем
- Юпитер (диспозитор) в Раке в 10 доме
- Луна в трине с Марсом

**Расчёт:**
```
Essential: Луна в Тельце (обитель) = +4
Accidental: Angular = +5, Free from combustion = +1
Dispositor: Юпитер в Раке (экзальтация) = +3
Aspects: Трин с Марсом = +2
House: Angular = +5

Total = 4 + 6 + 3 + 2 + 5 = 20
Category: Very Strong
```
