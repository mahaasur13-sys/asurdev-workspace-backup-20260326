# House Classification — Типы домов

Теги: #манкаси #дома #классификация #угловой #последующий #падающий

## 4 Quadruplicities (Качества)

### Angular (Угловые дома)

**Дома:** 1, 4, 7, 10

| Характеристика | Описание |
|----------------|----------|
| Сигнификатор | Активное начало, инициатива |
| Энергия | Самая сильная |
| Качество | Динамичная, концентрированная |
| Время | Быстрое проявление |
| Пример | 1 дом — "Кто я", 10 дом — "Что я делаю" |

### Succedent (Последующие дома)

**Дома:** 2, 5, 8, 11

| Характеристика | Описание |
|----------------|----------|
| Сигнификатор | Поддержка, стабильность |
| Энергия | Средняя |
| Качество | Устойчивая, накопительная |
| Время | Среднее проявление |
| Пример | 2 дом — ресурсы, 11 дом — надежды |

### Cadent (Падающие дома)

**Дома:** 3, 6, 9, 12

| Характеристика | Описание |
|----------------|----------|
| Сигнификатор | Служение, адаптация |
| Энергия | Слабая |
| Качество | Передаточная, рассеивающая |
| Время | Медленное проявление |
| Пример | 3 дом — коммуникация, 12 дом — изоляция |

## House Strength Modifiers

| Тип дома | Essential Dignity Modifier |
|----------|---------------------------|
| Angular | +4 |
| Succedent | +2 |
| Cadent | +1 |

## Дополнительные классификации

### Angular + Succedent (Kite Pattern)

Если планеты формируют воздушный змей с угловым домом как центром.

### Grand Trine

Три планеты в тригоне создают автоматическую силу для этих домов.

### T-Square

Напряжённая конфигурация с фокусом в одном доме.

### Bucket

5 планет в одной половине круга, одна в противоположной.
