# Мухурта — Путешествия (#мухурта #путешествие)

## Благоприятные условия для путешествий

### По положению Луны
- **Растущая Луна (Шукла Пакша)** — лучшее время для новых начинаний и поездок
- **Убывающая Луна (Кришна Пакша)** — подходит для возвращения домой

### По дням недели
| День | Тип путешествия | Рекомендация |
|------|------------------|--------------|
| Воскресенье | К Shor | Короткие поездки |
| Понедельник | Водный | Благоприятно |
| Вторник | Сухопутное | Осторожно |
| Среда | Воздушное | Благоприятно |
| Четверг | Любое | Лучший день |
| Пятница | К близким | Благоприятно |
| Суббота | Дальние | Избегать |

## Йоги Победы в Путешествии

Согласно [[Йоги Победы в Путешествии]], благоприятные комбинации:
- Луна в [[Мула]] или [[Джьештха]] + благоприятный асцендент
- Солнце в экзальтации или собственных знаках
- Марс в [[Карка]] или [[Макара]]

## Запреты (#запрет)

- **Бхадра** — первая мухурта любого дня (особенно для долгих поездок)
- [[Мула]] — 4-я пада (неблагоприятна для возвращения)
- [[Ашлеша]] — 4-я пада (опасно для水路)
- Время [[Калахора]] — солнечные затмения

## Рекомендации по направлению

- [[Дик-Шула]] — направление, которого следует избегать
- Движение против [[Дик-Шула]] — благоприятно
- Воскресенье: восток
- Понедельник: северо-запад
- Вторник: юг
- Среда: северо-восток
- Четверг: север
- Пятница: восток
- Суббота: запад
