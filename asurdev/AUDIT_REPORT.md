## Ключевые находки

### КРИТИЧНО
- **Нет гарантии вызова эфемерид** - агент может пропустить вызов и использовать знания LLM
- **Нет retry/fallback** - при сбое DE421 загрузки система падает
- **Нет unit-тестов** - невозможно отследить поломки
- **PII в логах** - даты рождения и координаты не анонимизированы

### СРЕДНЕ
- Нет кэширования по JD_UT - 30MB DE421 загружается при каждом запросе
- Нет параметра ayanamsa - нельзя переключить Lahiri/Raman
- Запрет на интерпретации между агентами не формализован

## Блок 1: Структура агентов

**Статус:** Частично

- MerrimanAgent, CycleAgent без описания роли

## Блок 2: Обязательный вызов эфемерид

**Статус:** ❌ НЕТ ENFORCEMENT

AstroCouncilAgent._calculate_positions() использует Skyfield, НО:
- Нет гарантии что вызовется
- Western/Vedic агенты получают позиции как параметр
- Fallback возвращает нули (тихая ошибка)

**Рекомендация:**
```python
@require_ephemeris
async def analyze(self, context):
    # Принудительный вызов если не передан positions
    if not context.get("positions"):
        context["positions"] = self._calculate_positions(dt)
```

## Блок 7: Безопасность

**Статус:** ❌ Критично
