# Мухурта: Результаты Транзитов и Начало Главы о Санскарах

Эта заметка завершает главу о транзитах (`Гочара`), предоставляя ключевые правила о времени проявления планетарных результатов и методах их коррекции. Затем она открывает новую главу, посвященную `Санскарам` (таинствам), начиная с правил для определения благоприятности первой менструации.

---

## 1. Дакшина (Пожертвования) для Планет (Продолжение)

Для умилостивления планет и привлечения их благосклонности рекомендуются следующие пожертвования (`Дакшина`).

| Планета | Рекомендуемое Пожертвование (`Дакшина`) |
| :--- | :--- |
| **Солнце** | Красный бык |
| **Луна** | Раковина |
| **Марс** | Красный бык |
| **Меркурий** | Золото |
| **Юпитер** | Желтая одежда (`Питамбара`) |
| **Венера** | Белая лошадь |
| **Сатурн** | Черная корова |
| **Раху** | Меч |
| **Кету** | Коза |

---

## 2. Временные Аспекты Результатов Транзитов

### Когда Планеты Начинают Давать Результаты *Следующего* Знака

Планеты начинают влиять энергией знака, в который они собираются войти, еще до самого перехода.

| Планета | Период до входа в следующий знак |
| :--- | :--- |
| **Солнце** | 5 дней |
| **Луна** | 3 `Гхати` (≈ 72 минуты) |
| **Марс** | 8 дней |
| **Меркурий** | 7 дней |
| **Юпитер** | 2 месяца |
| **Венера** | 7 дней |
| **Сатурн** | 6 месяцев |
| **Раху** | 3 месяца |

### Когда Планеты Дают Результаты *Внутри* Знака

Результаты транзита проявляются наиболее сильно в определенной части знака (разделенного на 3 части по 10 градусов — деканаты).

| Планета | Активный Период внутри Знака |
| :--- | :--- |
| **Солнце и Марс** | **Начало** (первый деканат: 0-10°) |
| **Юпитер и Венера** | **Середина** (второй деканат: 10-20°) |
| **Луна и Сатурн** | **Конец** (третий деканат: 20-30°) |
| **Меркурий** | **Всегда** (на протяжении всего знака) |

---

## 3. Упайи для Неблагоприятных Факторов в Мухурте

Если при выборе времени для важного дела (например, путешествия) некоторые астрологические факторы неблагоприятны, их можно нейтрализовать пожертвованиями.

| Неблагоприятный Фактор | Рекомендуемое Пожертвование |
| :--- | :--- |
| **Йога** | Золото |
| **Чандра (Луна)** | Раковина |
| **Карана** | Зерно |
| **Титхи (Лунный день)** | Рис (`Тандула`) |
| **Вара (День недели)** | Драгоценный камень |
| **Раши (Знак)** | Корова |
| **Нади / Лагна** | Золото |
| **Тара (Накшатра)** | Каменная соль (`Синдхуттхам`) |

---

## 4. Месячный Прогноз по Дню Входа Джанма-Накшатры

Определите, на какой день недели выпадает ваша натальная накшатра (`Джанма Накшатра`) в текущем лунном месяце, чтобы получить прогноз на этот месяц.

| День недели, на который выпала Джанма-Накшатра | Результат на Месяц |
| :--- | :--- |
| **Воскресенье** | `Адхван` (Путешествия, ходьба) |
| **Понедельник** | `Навахни` (Получение хорошей еды) |
| **Вторник** | `Бхайя` (Страх огня) |
| **Среда** | `Санмати` (Хорошие мысли, обучение) |
| **Четверг** | `Вастра` (Получение одежды) |
| **Пятница** | `Саукхья` (Счастье, комфорт) |
| **Суббота** | `Дуккхани` (Печаль, страдания) |

# Начало Главы: Санскара Пракаранам (Глава о Таинствах)

## 1. Первые Менструации (`ऋतु शुद्धि` / Ritu Shuddhi / Adhyam Rajah)

Первая менструация является важной `Санскарой`, и время, когда она происходит, влияет на будущую жизнь и здоровье женщины.

###
Благоприятные Условия (`Шубхам`)

- **Месяцы**: Магха, Маргаширша (Аграхаяна), Вайшакха, Ашвин, Пхалгуна, Джьештха, Шравана.
- **Пакша (фаза Луны)**: `Шукла Пакша` (растущая Луна).
- **Вара (день недели)**: Благоприятный день (`Сат-вара`).
- **Лагна (Асцендент)**: Благоприятный Асцендент (`Сат-танау`), аспектированный или соединенный с бенефиками.
- **Время суток**: День (`Дива`).

### Классификация Накшатр

| Уровень | Накшатры | Тип Накшатр (пояснение) |
| :--- | :--- | :--- |
| ✅ **Лучшие** | Шравана, Дхаништха, Шатабхиша, Мригашира, Читра, Анурадха, Ревати, Рохини, 3 Уттары, Свати | `Шрутитраям`, `Мриду` (мягкие), `Дхрува` (неподвижные), `Кшипра` (быстрые) |
| 🟡 **Средние**| Мула, Пунарвасу | `Мула` и `Адити` (накшатра Пунарвасу) |
| ❌ **Худшие** | Магха, Бхарани, Криттика, Ашлеша, Джиештха, Пурва Пхалгуни, Пурва Ашадха, Пурва Бхадрапада, Ардра, Ашвини, Вишакха | `Питри` (Магха), `Мишра` (смешанные), и остальные (`Парешу`) |
