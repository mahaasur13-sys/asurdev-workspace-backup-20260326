# Неблагоприятные периоды в ведической астрологии

#ведическая-астрология #мухурта #неблагоприятные-дни #йоги

## ❌ Периоды, которых следует избегать во всех благоприятных делах

Согласно ведическим текстам, следующие периоды считаются неблагоприятными для начала важных дел:

### 1. Основные неблагоприятные факторы

- **Джанма-накшатра** - накшатра рождения
    
- **Дата рождения**
    
- **Вьятипат-йога**, **Бхадра**, **Вайдхрити-йога**
    
- **Амавасья** - новолуние
    
- **Дата смерти родителей**
    
- **Кшая-титхи** - разрушительные лунные дни
    
- **Вриддхи-титхи** - определенные титхи
    
- **Кшаямас**, **Адхимас** - особые месяцы
    
- **Кулик**, **Ардхайама**, **Махапат-йога**
    

### 2. Запрещенные йоги и их отрезки

|Йога|Запрещенный период (в строфах/гхати)|
|---|---|
|**Вишкумбхи**|3 стиха|
|**Ваджра**|3 стиха|
|**Паригха**|первая половина|
|**Шула**|первые 5 стихов|
|**Ганда** и **Атиганда**|6 стихов|
|**Вьягхат**|9 стихов|

## 📅 Пакшарандхра-титхи - неблагоприятные лунные дни

### Запрещенные титхи:

- **Чаути** (4-й лунный день)
    
- **Чхати** (6-й лунный день)
    
- **Аштами** (8-й лунный день)
    
- **Навами** (9-й лунный день)
    
- **Двадаши** (12-й лунный день)
    
- **Чатурдаши** (14-й лунный день)
    

### Допустимые отрезки для срочных дел:

|Титхи|Максимальный допустимый период (данда)|
|---|---|
|Чаути|8 данд|
|Чхати|9 данд|
|Аштами|14 данд|
|Навами|24 данд|
|Двадаши|10 данд|
|Чатурдаши|5 данд|

## ⏳ Расчет Кулика-доши и подобных периодов

### Методика расчета:

1. **Определите текущий день недели**
    
2. **Посчитайте количество дней до целевого дня**:
    
    - Для **Кулика** - до субботы
        
    - Для **Калвела** - до среды
        
    - Для **Ямагант** - до четверга (Юпитер)
        
    - Для **Кантак** - до вторника
        
3. **Умножьте количество дней на 2** - получите номер неблагоприятной мухурты
    

### Пример для воскресенья:

|Доша|День назначения|Количество дней|Расчет|Неблагоприятная мухурта|
|---|---|---|---|---|
|**Кулика**|Суббота|7|7 × 2 = 14|14-я мухурта|
|**Калвела**|Среда|4|4 × 2 = 8|8-я мухурта|
|**Ямагант**|Четверг|5|5 × 2 = 10|10-я мухурта|
|**Кантак**|Вторник|3|3 × 2 = 6|6-я мухурта|

### Последствия действий в эти периоды:

- **Кулика мухурта** - полное разрушение
    
- **Ямагханта** - бедность
    
- **Калвела** - смерть
    
- **Кантак** - препятствия
    

> **Важно**: Эти доши не действуют в ночное время. Для очень важных дел можно использовать только последнюю часть этих периодов.
