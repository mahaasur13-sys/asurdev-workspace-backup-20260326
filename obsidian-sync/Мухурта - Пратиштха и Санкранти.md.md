# Мухурта: Пратиштха (Освящение) и Санкранти (Переход Солнца)

Эта заметка охватывает правила [[Мухурта|Мухурты]] для освящения и установки (Пратиштха) божеств, храмов, садов и водоемов. Также здесь начинается новая глава, посвященная классификации и результатам Санкранти — момента перехода Солнца из одного знака в другой.

---

## 🙏 Пратиштха: Освящение Божеств, Садов и Водоемов

### ✅ Общие благоприятные условия
- *Аяна*: [[Уттараяна]] (северный ход Солнца).
- *Видимость планет*: [[Юпитер]], [[Луна]] и [[Венера]] должны быть видимы (не сожжены).
- *Типы [[Накшатры|Накшатр]]*:
    - [[Мягкие накшатры (Мриду)|Мриду]]
    - [[Быстрые накшатры (Кшипра)|Кшипра]]
    - [[Подвижные накшатры (Чара)|Чара]]
    - [[Фиксированные накшатры (Дхрува)|Дхрува]]
- *Пакша*: [[Шукла Пакша]] (растущая Луна).
- *Титхи и дни*: Следует избегать [[Рикта Титхи]] и Вторника.
- *Особое правило*: Также благоприятно использовать накшатру и титхи, связанные с самим божеством, которое устанавливается.

### ✅ Требования к Лагне (Асценденту)
- *Пагубные планеты*: Должны находиться в 3, 6 или 11 домах от Лагны (дома Упачая).
- *Благоприятные планеты*: Должны находиться в Кендрах (1, 4, 7, 10) или Конах (5, 9).
    > [!WARNING] Избегать 8-го и 12-го домов для благоприятных планет.
- *Тип Лагны*: Фиксированный ([[Стхира Раши]]) или двойственный ([[Двисвабхава Раши]]) знак.

| :--- | :--- | :--- |
| *Солнце (Сурья)* | [[Лев]] | [[Хаста]] |
| *Вишну* | [[Дева]] | - |
| *Шива* | [[Близнецы]] | - |
| *Брахма* | [[Водолей]] | - |
| *Дэви (Богиня)* | Двойственные знаки | - |
| *Кшудра (Малые божества, Йогини)* | Подвижные знаки | - |
| *Ганеша, Якши, Наги, Бхуты* | - | [[Ревати]] |
| *Будда (Джина)* | - | [[Шраван]] |
| *Луна и все 9 планет (Наваграхи)*| - | [[Пушья]] |
| *Все божества (общее правило)* | Фиксированные знаки | - |

## ☀️ Санкранти: Переход Солнца и его Результаты

Имя и результат каждого [[Санкранти]] определяются комбинацией дня недели и накшатры, в которой находится Солнце в момент перехода в новый знак.

| Имя Санкранти | День недели | Накшатры | Результат (Кому приносит счастье) |
| :--- | :--- | :--- | :--- |
| *Гхора* (Ужасающая) | Воскресенье | [[Пурва Пхалгуни]], [[Пурвашадха]], [[Пурвабхадрапада]], [[Бхарани]], [[Магха]] | Шудрам (рабочим) |
| *Дхванкши* (Воронья) | Понедельник | [[Хаста]], [[Ашвини]], [[Пушья]], [[Абхиджит]] | Вайшьям (торговцам) |
| *Маходари* (Большебрюхая)| Вторник | [[Свати]], [[Пунарвасу]], [[Шраван]], [[Дхаништха]], [[Шатабхиша]] | Ворам |
| *Мандакини* | Среда | [[Мригашира]], [[Ревати]], [[Читра]], [[Анурадха]] | Кшатриям (правителям, воинам) |
| *Манда* (Медленная) | Четверг | [[Уттара Пхалгуни]], [[Уттарашадха]], [[Уттарабхадрапада]], [[Рохини]] | Брахманам (ученым, священникам) |
| *Мишра* (Смешанная) | Пятница | [[Вишакха]], [[Криттика]] | Животным |
| *Ракшаси* (Демоническая)| Суббота | [[Мула]], [[Джьештха]], [[Ардра]], [[Ашлеша]] | Чандалам (низшим кастам) |
