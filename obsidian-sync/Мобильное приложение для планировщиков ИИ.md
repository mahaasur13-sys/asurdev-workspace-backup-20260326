**Заметка: Мобильное приложение для планировщиков ИИ (Fast Downward, POPF, TFD, OPTIC, LAMA, Val)**

**Архитектура:** Клиент-серверная модель. Тонкий клиент (мобильное приложение) для ввода задач и визуализации, сервер — для тяжелых вычислений (генерация планов).

**Ключевые компоненты:**

- 🖥️ **Клиент:** React Native/Flutter, интерфейс для загрузки PDDL/HDDL, визуализация планов (DisneyViz, графы).
    
- 🌐 **Сервер:** Python/Node.js, обёртки для планировщиков (Planutils, Docker), REST API/WebSocket.
    
- 🗄️ **БД:** PostgreSQL/MongoDB для хранения планов и доменов.
    

**Технологии:** Docker, Kubernetes, AWS/GCP, ROS2 (для интеграции с робототехникой).

**Вызовы:**

- Производительность (оптимизация под мобильные устройства).
    
- Безопасность (HTTPS, аутентификация).
    
- Синхронизация данных (офлайн-режим с кэшированием).
    

**Применение:** Образование (обучение планированию), промышленность (управление роботами), исследовательские проекты.
