# Мухурта: Предсказания по Санкранти (Карана и Накшатра)

Эта заметка содержит подробное описание `Санкранти Пуруша` — персонифицированного образа перехода Солнца — для каждой из одиннадцати `Каран`. Также здесь представлена техника личного прогнозирования на месяц, основанная на положении натальной накшатры относительно накшатры, в которой происходит Санкранти.

---

## ব্যক্তিগত Прогноз на месяц по Накшатре

Это простой метод для определения личных результатов на месяц, основанный на положении вашей натальной накшатры (`Джанма Накшатра`) относительно накшатры, в которой произошел Санкранти.

*Метод*: Сосчитайте количество накшатр, начиная от накшатры Санкранти до вашей натальной накшатры (включительно).

| Позиция от Санкранти Накшатры | Результат |
| :--- | :--- |
| *1 — 3* | *Гаманам* (Путешествия, перемещения) |
| *4 — 9* | *Сукхам* (Счастье, комфорт) |
| *10 — 12*| *Пиданам* (Страдания, боль) |
| *13 — 18*| *Аншукам* (Потеря одежды/статуса, бесчестье) |
| *19 — 21*| *Артхахани* (Потеря богатства) |
| *22 — 27*| *Дханагама* (Прибыль, приход богатства) |

> [!TIP] Пример расчета
> - *Предположим*, Санкранти происходит в накшатре *Криттика*.
> - *Ваша натальная накшатра* — *Хаста*.
> - *Считаем*: 1. Криттика, 2. Рохини, 3. Мригашира, 4. Ардра, 5. Пунарвасу, 6. Пушья, 7. Ашлеша, 8. Магха, 9. Пурва Пхалгуни, 10. Уттара Пхалгуни, *11. Хаста*.
> - *Результат*: Ваша накшатра находится на 11-й позиции. Смотрим в таблицу: позиция 11 попадает в диапазон *10-12*, что дает результат *Пиданам* (Страдания, боль) на этот месяц.
