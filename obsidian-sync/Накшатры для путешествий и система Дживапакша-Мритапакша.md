# Накшатры для путешествий и система Дживапакша-Мритапакша

## ⏰ Запрещённые периоды для путешествий

### Ночное деление и запрещённые накшатры:

| Период ночи | Запрещённые накшатры |
| :---------- | :------------------- |
| **Первая часть** | Ревати, Читра, Анурадха |
| **Вторая часть** | Пурва (Пурва-Пхалгуни, Пурва-Ашадха, Пурва-Бхадрапада), Бхарани, Магха |
| **Третья часть** | Свати, Пунарвасу, Дхаништха, Шатабхиша |

### Всегда благоприятные накшатры:
- **Шраван**
- **Хаста** 
- **Пушья**
- **Мригашира**

> Путешествие в этих накшатрах благоприятно в любое время.

## ⚠️ Запрещённые данды в накшатрах

### Критические периоды:

| Накшатра | Запрещённые данды |
| :------- | :---------------- |
| **Три Пурва** | Первые 16 данд |
| **Криттика** | 21 данда |
| **Бхарани** | 7 данд |
| **Свати, Вишакха, Джьештха, Ашлеша** | 14 данд |

## 👑 Особые указания для царей

### По разным мнениям:

**Общее правило:**
- **Первая половина** Магхи и Свати
- **Последняя часть** Читры, Ашлеши, Бхарани

**Мнение Шукрачарьи:**
- Полностью избегать **Свати** и **Магхи** во время путешествия

## 🔄 Система Дживапакша-Мритапакша

### Основные понятия:

| Термин | Значение |
| :----- | :------- |
| **Дживапакша** | "Живая сторона" - 13 благоприятных накшатр |
| **Мритапакша** | "Мёртвая сторона" - 13 неблагоприятных накшатр |
| **Картари-санкхьяка** | Накшатра, где находится Раху |

### Механизм определения:

1. **Определяется положение Раху**
2. **Отсчитываются 13 накшатр в обратном порядке** - Дживапакша
3. **Следующие 13 накшатр** - Мритапакша
4. **Накшатра Раху** - Картари-санкхьяка
5. **15-я накшатра от Раху** - поражаемая

### Пример:
Если **Раху в Хасте**:
- **Дживапакша**: Читра → Пурва-Бхадрапада (13 накшатр)
- **Мритапакша**: Уттара-Пхалгуни → Пурва-Ревати (13 накшатр) 
- **Картари**: Хаста
- **Поражённая**: Уттара-Бхадрапада

## ☀️ Влияние Солнца на Дживапакшу

### Особые случаи:

| Положение Солнца | Рекомендация |
| :--------------- | :---------- |
| **В Мритапакше** | Путешествие в Дживапакшу благоприятно |
| **В Дживапакше** | Путешествие в Мритапакшу разрушительно |
| **Две Дживапакши** | Путешествие благоприятно |
| **Прямое наблюдение Мритапакши** | Может быть благоприятным |
| **Затменная Картари** | Благоприятна для путешествия |
| **Луна в Дживапакше** | Победоносное путешествие |
