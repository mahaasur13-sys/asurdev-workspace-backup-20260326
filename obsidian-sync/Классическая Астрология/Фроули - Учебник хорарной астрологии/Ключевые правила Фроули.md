# Ключевые правила хорарной астрологии по Фроули

## Правила суждения

### 1. Планеты показывают, а не решают
- Астролог читает карту
- Не привносит субъективность
- Факты важнее интерпретаций

### 2. Вопрос определяет толкование
- Один вопрос — одна тема
- Нельзя смешивать темы
- Контекст вопроса критичен

### 3. Сигнификаторы
| Показатель | Значение |
|------------|----------|
| **Кверент** | ASC + его управитель |
| **Вопрошающий** | Солнце (иногда) |
| **Предмет вопроса** | Управитель дома вопроса |

### 4. Сила планеты
- В собственной обители — максимальная сила
- В экзальтации — высокая сила
- В изгнании — слабость
- В падении — минимальная сила

### 5. Аспекты к сигнификаторам
- Соединение — связь
- Квадрат — напряжение
- Трин — гармония
- Оппозиция — конфликт/дилемма

## Уникальный вклад Фроули

### Холистический подход
Фроули настаивает на рассмотрении **всей карты**, а не отдельных показателей.

### Принцип «Одна карта — один вопрос»
Нарушение этого правила — главная ошибка начинающих.

### Важность Лунного знака
Луна показывает **текущее состояние** и движение энергии.

### Рецепция — ключ к пониманию
Взаимное расположение планет по достоинству показывает **истинные отношения** между факторами.
