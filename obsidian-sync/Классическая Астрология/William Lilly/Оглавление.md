# Christian Astrology
## William Lilly, 1647

### О книге
Классический труд по астрологии, написанный английским астрологом William Lilly в 1647 году. Один из наиболее влиятельных учебников по астрологии в англоязычном мире.

### Структура
- **Book 1**: An Introduction to Astrology (Введение в астрологию)
- **Book 2**: The Resolution of All Manner of Questions (Решение всех видов вопросов)
- **Book 3**: Horary Questions (Горальная астрология)

### Ключевые концепции
- #тригон-огненный = творческий, сильный
- #тригон-земной = практический, стабильный
- #тригон-воздушный = социальный, интеллектуальный
- #тригон-водный = эмоциональный, интуитивный

### Мухурта по Lilly
Lilly описывает систему определения благоприятного времени для действий, аналогичную индийской Мухурте:

| Время | Качество | Описание |
|-------|----------|----------|
| Утренние часы | Лучшее | Для духовных практик |
| Полуденные | Хорошее | Для деловых вопросов |
| Вечерние | Среднее | Для социальных дел |
| Ночные | Осторожно | Для тайных операций |
