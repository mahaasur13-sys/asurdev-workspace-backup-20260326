# Electional Astrology
## Искусство выбора благоприятного времени

> *"Не начинайте важного дела, пока не убедитесь, что небеса благосклонны."*  
> — William Lilly

---

## Основные принципы элекции

### 1. Управитель дня
| День | Управитель | Лучше для |
|------|-----------|-----------|
| Воскресенье | Солнце | Церемонии, коронации |
| Понедельник | Луна | Начинания, путешествия |
| Вторник | Марс | Войны, конфликты |
| Среда | Меркурий | Торговля, переговоры |
| Четверг | Юпитер | Религия, образование |
| Пятница | Венера | Любовь, брак, искусство |
| Суббота | Сатурн | Строительство, картагенез |

### 2. Управитель часа
Каждый час управляется планетой в порядке:
```
Солнце → Венера → Меркурий → Луна → Сатурн → Юпитер → Марс → (повтор)
```

| Знак | Лучше для | Хуже для |
|------|-----------|----------|
| Aries | Начинания, война | Партнёрство |
| Taurus | Сельское хозяйство, деньги | Перемены |
| Gemini | Общение, торговля | Серьёзные дела |
| Cancer | Дом, семья | Конфликты |
| Leo | Церемонии, творчество | Тайные дела |
| Virgo | Здоровье, работа | Любовь |
| Libra | Партнёрство, баланс | Начинания |
| Scorpio | Трансформация,秘密 | Повседневность |

| Capricorn | Карьера, структура | Эмоции |
| Aquarius | Новаторство, группы | Традиции |
| Pisces | Духовность, фантазии | Юридические дела |

## Лучшие времена для действий

### ✅ ИДЕАЛЬНОЕ время:
1. Луна в знаке своей экзальтации (Cancer)
2. Луна в тригоне к Солнцу и Юпитеру
3. Управитель часа — Юпитер или Венера
4. Нет аспектов от Марса или Сатурна

### ❌ ИЗБЕГАТЬ времени:
1. **Безлуние** (Новолуние ± 3 дня)
2. **Солнечные/Лунные затмения** (± 14 дней)
3. **Ретроградный Меркурий** (для новых начинаний)
4. **Марс в огненных знаках** (для мирных дел)
