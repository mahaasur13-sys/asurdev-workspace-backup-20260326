Амрит Сиддхи Йога и Сарва Сиддхи Йога: свойства и концептуальная интеграция Метаданные (YAML для Obsidian)


ОбзорАмрит Сиддхи Йога и Сарва Сиддхи Йога — это благоприятные комбинации в ведической астрологии, созданные для обеспечения успеха, долговечности начинаний и исполнения желаний.  
В этой заметке свойства этих йог семантически переосмыслены и сжаты в современную концепцию, где акцент смещён с вопроса «как делать» на вопрос «что создавать» — это и есть ключевая идея демократизации технологий.

[!note] Главная суть  
Фокус смещается с процедурного «как правильно сделать» на визионерское «что именно стоит создать», что открывает доступ к созиданию для всех.

Свойства Амрит Сиддхи Йоги

- Буквальный смысл: Йога бессмертия / вечного достижения (Амрит = нектар бессмертия, Сиддхи = достижение, совершенство).
- Формирование: Определённые комбинации дня недели и накшатры (например, определённые звёзды в определённые дни дают максимальную силу благоприятного направления).
- Преимущества:
    - Гарантирует положительный результат начатых дел, проектов, намерений
    - Идеальна для любовных браков, дальних путешествий и важных жизненных событий
    - В контексте Хатха-йоги (текст Амритасиддхи): контроль бинду через махамудру и другие практики → физическое и духовное бессмертие
- Благоприятные действия: начало проектов, которым нужна долговечность и устойчивый успех
- Концептуальная связь: Создание «бессмертных» цифровых артефактов (open-source, вечнозелёные инструменты, которые продолжают приносить пользу людям десятилетиями).

Свойства Сарва Сиддхи Йоги (Сарвартха Сиддхи Йоги)

- Буквальный смысл: Достижение всех целей / исполнение всех желаний (Сарва = всё, Артха = цели/желания, Сиддхи = успех).
- Формирование: Специфические комбинации планет, дней недели и накшатр.
- Преимущества:
    - Подходит практически для любого начинания
    - Исполняет желания, устраняет препятствия
    - Нейтрализует негативное влияние других неблагоприятных йог
- Концептуальная связь: Демократизация созидания — любой человек может реализовать свою идею с помощью доступных современных инструментов.

Ключевая концепция: Цикл Сиддхи в эпоху демократизации технологийЦентральный элемент — итеративный цикл Сиддхи:  
запрос → реализация → обратная связь → исправлениеИменно этот цикл обеспечивает «амрит» (бессмертие результата) и «сарва» (всеобъемлющий успех).

- Запрос — формулируем видение («Хочу создать инструмент ИИ для персональной йога-практики»)
- Реализация — воплощаем через доступные технологии (no-code, low-code, промпт-инжиниринг)
- Обратная связь — собираем реакции пользователей, сообщества
- Исправление — улучшаем, делаем лучше и долговечнее

[!tip] Простой пример применения  
Переход от изучения «как кодить сайт» → к вопросу «что именно я хочу создать для людей?»  
Это и есть демократизация: любой может создавать ценность, не будучи профессиональным программистом.
