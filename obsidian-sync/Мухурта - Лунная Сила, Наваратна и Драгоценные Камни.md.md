# Мухурта: Сила Луны по Пакше и Наваратна (Драгоценные Камни)

Эта заметка охватывает два важных аспекта практической астрологии. Во-первых, это особое правило для оценки силы Луны (`Абджабалам`) на целый лунный двухнедельный период, предназначенное для экстренных случаев. Во-вторых, здесь приводится классическая система `Наваратна` — девяти драгоценных камней, используемых для умиротворения и гармонизации планетарных энергий.

---

## 🌙 Сила Луны в Зависимости от Начала Пакши (для Кризисных Ситуаций)

Это правило используется, когда необходимо совершить важное действие (брак, путешествие) в срочном порядке, и нет возможности дождаться идеального `Чандра Бала` (силы Луны по транзиту). Сила или слабость Луны в первый день (`Пратипада`) лунной половины (`Пакши`) определяет ее характер на все последующие 15 дней.

> [!INFO] Принцип "Противоположности" для Кришна Пакши
> Обратите внимание, что для темной половины Луны (Кришна Пакша) действует обратный принцип.

| Состояние Луны в 1-й день (Пратипада) | Затронутая Пакша (15 дней) | Результат на всю Пакшу |
| :--- | :--- | :--- |
| *Благоприятная* Луна | *Шукла Пакша* (Растущая) | *Благоприятно* |
| *Неблагоприятная* Луна | *Кришна Пакша* (Убывающая) | *Благоприятно* |
| *Неблагоприятная* Луна | *Шукла Пакша* (Растущая) | *Неблагоприятно* |
| *Благоприятная* Луна | *Кришна Пакша* (Убывающая) | *Неблагоприятно* |

## 💎 Наваратна: Драгоценные Камни для Планетарной Гармонии

Для умиротворения планет (`Граха Шанти`) рекомендуется носить определенные драгоценные камни, вставленные в золотую янтру (пластину) из девяти ячеек в соответствии с их направлением.

### Расположение камней в Янтре (вид сверху)

| | | |
| :---: | :---: | :---: |
| *С-В / NE* <br> Меркурий <br> Изумруд | *Север / N* <br> Юпитер <br> Желтый Сапфир (Топаз) | *С-З / NW* <br> Кету <br> Кошачий Глаз (Вайдурья) |
| *Восток / E* <br> Венера <br> Алмаз | *Центр* <br> Солнце <br> Рубин | *Запад / W* <br> Сатурн <br> Синий Сапфир |
| *Ю-В / SE* <br> Луна <br> Жемчуг | *Юг / S* <br> Марс <br> Красный Коралл | *Ю-З / SW* <br> Раху <br> Гессонит (Гомед) |
