# Мухурта - Яджняповита (Планетарные Влияния и Йоги)

Эта заметка является продолжением темы [[Яджняповита]] и рассматривает специфические результаты, основанные на положении планет в Навамше, домах Кендры и их соединениях в гороскопе Мухурты.

## Влияние Навамши Лагны

Результат церемонии сильно зависит от того, в чьей Навамше (дробная карта D9) находится [[Лагна]] (Асцендент) в момент проведения ритуала.

| Навамша Управителя | Результат для ребенка (Вату) |
| :--- | :--- |
| *Солнце* | Жестокий, бессердечный |
| *Луна* | Глупый, безрассудный |
| *Марс* | Грешный |
| *Меркурий* | Умный, сообразительный |
| *Юпитер* | Выполняющий шесть предписанных действий (изучение/обучение, дача/принятие даров, проведение/участие в яджнях) |
| *Венера* | Проводящий яджни и богатый |
| *Сатурн* | Глупый |

> *Вывод:* Лагна должна находиться в Навамше благоприятной планеты (Меркурия, Юпитера, Венеры).

## Влияние Навамши Луны

- *В Навамше благодетеля:* Ребенок будет всегда стремиться к знаниям.
- *В Навамше вредителя:* Станет очень бедным.
- *В своей Навамше (Рак):* Будет испытывать много страданий.

> *Исключение:* Если Луна находится в Навамше Рака, но церемония проходит в накшатру [[Шравана]] или [[Пунарвасу]], ребенок становится богатым.

## Результаты Планет в Кендре (1, 4, 7, 10 дома)

Планеты, расположенные в угловых домах гороскопа Мухурты, определяют будущую деятельность ребенка.

| Планета в Кендре | Результат (профессия, характер) |
| :--- | :--- |
| *Солнце* | Слуга правителя (госслужащий) |
| *Луна* | Торговец, бизнесмен (деятельность Вайшьи) |
| *Марс* | Военный, человек, живущий оружием |
| *Меркурий* | Учитель, лектор |
| *Юпитер* | Ученый, пандит, мудрец |
| *Венера* | Богатый человек |
| *Сатурн* | Слуга иноземцев (млеччхов) |

## Результаты Соединений с Благоприятными Планетами

Если Венера, Юпитер или Луна в момент Мухурты находятся в соединении с другими планетами, это дает следующие результаты:

- *Соединение с Солнцем:* Ребенок будет лишен хороших качеств.
- *Соединение с Марсом:* Будет жестоким в своих поступках.
- *Соединение с Сатурном:* Станет бесстыдным.
- *Соединение с Благоприятной планетой:* Будет искусен во всех науках.

## Особые Йоги для Знания

### Йога Знатока Всех Вед

Ребенок станет знатоком всех Вед, если в гороскопе Мухурты одновременно выполняются три условия:
1. *Луна* находится в Навамше *Венеры*.
2. *Венера* находится в доме [[Трикона]] (5-м или 9-м).
3. *Юпитер* находится в [[Лагне]] (1-м доме).

### Неблагоприятная Йога (Луна в Навамше Сатурна)

Если *Луна* находится в Навамше [[Сатурна]], ребенок становится очень жестоким и бессердечным.
