# Мухурта: Випарита Ведха, Разные Мнения и Результаты Затмений

Эта заметка раскрывает концепцию `Випарита Ведха` (Обратного Поражения), где неблагоприятный транзит может стать благоприятным. Также рассматриваются различные подходы к расчету Ведхи и приводятся подробные результаты затмений (`Грахана Пхалам`) в зависимости от их положения относительно натальной карты.

---

## 🔄 Випарита Ведха (Обратное Поражение)

В отличие от обычной [[Мухурта - Правила Ведхи в Гочаре.md|Крама Ведхи]], которая аннулирует хороший транзит, `Випарита Ведха` *превращает неблагоприятный транзит в благоприятный*.

*Принцип*: Если планета находится в неблагоприятном доме, но другая планета занимает соответствующий "поражающий" дом, то негативный результат транзита отменяется, и он становится благоприятным.

| Планета | Неблагоприятный дом транзита | Дом, создающий благоприятную Ведху |
| :--- | :--- | :--- |
| *Луна* | 4, 9, 8, 5, 12, 2 | 10, 3, 11, 1, 6, 7 |
| *Меркурий* | 5, 3, 9, 1, 8, 12 | 2, 4, 6, 8, 10, 11 |
| *Юпитер* | 4, 12, 10, 3, 8 | 5, 2, 9, 11, 7 |
| *Венера* | 8, 7, 1, 10, 9, 5, 11, 6, 3 | 1, 2, 4, 5, 8, 9, 12, 11, 1 |

> [!TIP] Особое правило для Луны в Шукла Пакша (Растущая фаза)
> Если Луна в Шукла Пакша находится в неблагоприятных 2, 5, 9 домах, она все равно дает хороший результат, *если* 4, 6, 8 дома свободны от планет.

---

### 🏛️ Разные Мнения о Применении Ведхи

Существуют разные подходы к тому, от какой точки отсчитывать Ведху.

1. *Крама Ведха (Прямое Поражение)*: Отсчитывается от `Джанма Раши` (натальной Луны). Это наиболее распространенный метод для определения *аннуляции хорошего транзита*. (Мнение Нарады).
2. *Випарита Ведха (Обратное Поражение)*: Отсчитывается от знака, в котором находится сама транзитная планета. Этот метод используется для определения *превращения плохого транзита в хороший*. (Мнение Кашьяпы).

> [!INFO] Географическое применение
> Мудрец Кашьяпа утверждает, что эти правила Ведхи наиболее сильны в центральных землях Индии (между Гималаями и горами Виндхья).

## 🌘 Грахана Пхалам (Результаты Затмений)

Результаты лунного или солнечного затмения зависят от его положения относительно вашей натальной карты.

> [!DANGER] Затмение в Джанма Накшатре
> Затмение, происходящее в вашей натальной накшатре (`Джанма Накшатра`), считается крайне неблагоприятным и может указывать на угрозу жизни (`Нидханам`).

### Результаты затмения по домам от Джанма Раши

| Дом от Джанма Раши, где происходит затмение | Результат |
| :--- | :--- |
| *1-й дом* | *Гхатах* (Удар, боль для тела) |
| *2-й дом* | *Хани* (Потеря, ущерб) |
| *3-й дом* | *Шри* (Богатство, процветание) |
| *4-й дом* | *Вьятха* (Страдание, горе) |
| *5-й дом* | *Чинта* (Беспокойство, тревога о детях) |
| *6-й дом* | *Саукхьям* (Счастье, комфорт) |
| *7-й дом* | *Калтрадаустьям* (Проблемы с супругом) |
| *8-й дом* | *Мритью* (Смерть, опасность для жизни) |
| *9-й дом* | *Мананаашах* (Потеря уважения, позор) |
| *10-й дом*| *Сукхам* (Счастье, успех) |
| *11-й дом*| *Лабхах* (Прибыль) |
| *12-й дом*| *Апаай* (Утрата, смерть) |

### 🙏 Шанти (Умиротворение) для неблагоприятных затмений

Для смягчения негативных последствий затмения рекомендуются следующие меры:

-   *Джапа*: Повторение мантр, особенно Махамритьюнджая-мантры ("Трьямбакам...").
-   *Дана*: Пожертвование золота или коровы.
-   *Не смотреть на затмение*: Некоторые мудрецы советуют физически не наблюдать за затмением, если оно неблагоприятно для вас.
