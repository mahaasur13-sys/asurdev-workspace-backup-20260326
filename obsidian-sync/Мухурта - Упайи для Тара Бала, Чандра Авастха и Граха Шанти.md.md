# Мухурта: Упайи для Тара Бала, Чандра Авастха и Граха Шанти

Эта заметка охватывает три ключевые практические темы:
1. **Упайи для Тара Доши**: Методы коррекции и нейтрализации негативного влияния неблагоприятных `Тар` (3-й, 5-й и 7-й).
2. **Чандра Авастха**: Расчет 12 состояний Луны для определения ее тонкого влияния в течение дня.
3. **Граха Шанти**: Общие и конкретные меры умиротворения планет через омовения и пожертвования.

---

## 1. Упайи (Меры Коррекции) для Неблагоприятных Тар

Если дневная накшатра попадает в неблагоприятную `Тару`, ее негативное влияние можно смягчить с помощью пожертвований (`Даана`).

| Неблагоприятная Тара (№) | Название | Рекомендуемое Пожертвование |
| :--- | :--- | :--- |
| **1-я** | Джанма | Овощи |
| **3-я** | Випат | Гур (Джаггери) |
| **5-я** | Пратьяри | Соль |
| **7-я** | Вадха | Золото и семена кунжута (`Тила`) |

### Продвинутая Нейтрализация: Правило Трех Кругов (`Парьяя`)

Для более тонкого анализа 27 накшатр делятся на три круга по 9. Негативность неблагоприятных `Тар` проявляется по-разному в каждом круге.

&gt; [!INFO] Как определить круг
&gt; - **Первый круг**: Накшатры с 1-й по 9-ю от вашей `Джанма Накшатры`.
&gt; - **Второй круг**: Накшатры с 10-й по 18-ю.
&gt; - **Третий круг**: Накшатры с 19-й по 27-ю.

| Круг | Влияние 3-й, 5-й и 7-й `Тар` |
| :--- | :--- |
| **Первый круг** | **Полностью неблагоприятны.** |
| **Второй круг** | **Частично неблагоприятны:**&lt;br&gt;- **3-я (Випат)**: Неблагоприятна **первая 1/3** накшатры.&lt;br&gt;- **5-я (Пратьяри)**: Неблагоприятна **средняя 1/3** накшатры.&lt;br&gt;- **7-я (Вадха)**: Неблагоприятна **последняя 1/3** накшатры. |
| **Третий круг** | **Считаются полностью благоприятными.** |

## 2. Чандра Авастха (12 Состояний Луны)

Это метод для определения одного из 12 "состояний" Луны, основанный на ее точном положении в зодиаке.

&gt; [!TIP] Расчет Чандра Авастха
&gt; 1. Возьмите номер пройденных накшатр от Ашвини (`Гатабхам`) и умножьте на 60.
&gt; 2. К результату прибавьте пройденные `Гхати` текущей накшатры (`Бхуктагхати`).
&gt; 3. Полученную сумму умножьте на 4.
&gt; 4. Разделите результат на 45.
&gt; 5. Целое число от деления (`Лабдха`) — это количество *пройденных* состояний. Следующее число указывает на *текущее* состояние, которое соответствует знаку зодиака.
&gt;     - _Пример: если результат 5, то 5 состояний пройдены, и сейчас идет 6-е состояние (Дева)._

### Названия и Результаты 12 Состояний

| Знак Зодиака (где Луна) | Название Состояния (Авастха) | Результат (Пхала) |
| :--- | :--- | :--- |
| 1. Овен | `Праваса` | Миграция, путешествие |
| 2. Телец | `Наша` | Разрушение, потеря |
| 3. Близнецы| `Мритью` | Смерть, опасность |
| 4. Рак | `Джая` | Победа |
| 5. Лев | `Хаса` | Смех, радость |
| 6. Дева | `Рати` | Удовольствие, игра |
| 7. Весы | `Крида` | Спорт, развлечения |
| 8. Скорпион| `Супта` | Сон, бездействие |
| 9. Стрелец | `Бхукта` | Страдание |
| 10. Козерог| `Джвара` | Лихорадка, болезнь |
| 11. Водолей| `Кампа` | Трепет, беспокойство |
| 12. Рыбы | `Стхирата` | Стабильность |

## 3. Граха Шанти (Умиротворение Планет)

### Лечебное Омовение
Для устранения негативного влияния **всех** планет рекомендуется омовение в воде, смешанной с лечебными травами:
Ладжавати, Кута, Барияра, Какуни, Муста (Сыть), Горчица, Куркума, Деодар, Шарпункха и
Лодхра.

### Дакшина (Пожертвования) для Планет
В тексте начинается перечисление рекомендуемых пожертвований (`Дакшина`) для умиротворения планет, начиная с Солнца.

| Планета (вероятно) | Рекомендуемое Пожертвование |
| :--- | :--- |
| **Юпитер** | Желтая одежда (`Питамбара`) |
| **Солнце/Марс**| Лошадь (`Гхотака`) |
| **Венера/Луна**| Белая корова (`Швето Гау`) |
| **Сатурн** | Черная (корова/одежда) (`Асита`) |
| **Сатурн/Раху**| Буйвол (`Махиша`) |
| **Марс** | Козел (`Аджа`) |
