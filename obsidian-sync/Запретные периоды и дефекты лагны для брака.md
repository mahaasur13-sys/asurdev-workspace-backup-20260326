# Запретные периоды и дефекты лагны для брака

## ⚠️ Запретный период во время Сурья Санкранти (Шлока 79)

**Санкранти** - это переход Солнца из одного знака зодиака в другой.

### Особо неблагоприятные Санкранти:
- **Вишува** - равноденствие (Весы и Овен)
- **Аяна** - солнцестояние (Рак и Козерог)

### Запретные дни:
| Период | Длительность | Что запрещено |
| :----- | :----------- | :------------ |
| **Вишува и Аяна Санкранти** | День Санкранти + 1 день до + 1 день после (всего 3 дня) | Все благоприятные дела, включая **брак** |
| **Другие Санкранти** | 16 гхатик до + 16 гхатик после (всего 32 гхатики) | Все благоприятные дела, включая **брак** |

> **Примечание:** 1 гхатика = 24 минуты, 16 гхатик = 6 часов 24 минуты

## 🪐 Запретные периоды планетных Санкранти (Шлока 80)

Во время переходов различных планет (Граха Санкранти) также существуют запретные периоды:

| Планета | Запретный период (в дандах) |
| :------ | :-------------------------- |
| **Солнце** (Сурья) | 33 данды |
| **Луна** (Чандра) | Навданда (9 данд?) |
| **Марс** (Мангал) | Навданда |
| **Меркурий** (Буддха) | 6 данд |
| **Юпитер** (Гуру) | 88 данд |
| **Венера** (Шукра) | Навданда |
| **Сатурн** (Шани) | 160 данд |

> **Особое внимание:** 33 данды периода Солнечной Санкранти считаются **очень неблагоприятными**.

## 👁️ Дефекты лагны - "Парализующая слепота" (Шлока 81-82)

### Дефекты по времени суток:

| Время | Дефект лагны | Знаки зодиака |
| :---- | :----------- | :------------ |
| **Днём** | Бадхира (Глухота) | Весы, Скорпион |
| **Ночью** | Бадхира (Глухота) | Козерог, Стрелец |
| **Днём** | Андха (Слепота) | Лев, Овен, Телец |
| **Ночью** | Андха (Слепота) | Дева, Близнецы, Рак |
| **Днём** | - | Водолей |
| **Ночью** | - | Рыбы |

### Дополнительные уточнения (Шлока 82):
- **Бадхира** (глухота): Стрелец, Весы, Скорпион (первые 2 часа)
- **Андха** (слепота): Близнецы, Рак, Дева (ночью); Лев, Телец, Овен (днём)
- **Кубджа** (искривление): Козерог, Водолей (утром и вечером)

## 💔 Результаты дефектных лагн для брака (Шлока 83)

| Дефект лагны | Последствия брака |
| :---------- | :---------------- |
| **Бадхира** (Глухота) | Бедность (Даридра) |
| **Андха днём** (Слепота) | Вдовство (для девушки) |
| **Андха ночью** (Слепота) | Бесплодие, отсутствие детей |
| **Кубджа** (Искривление) | Уничтожение богатства |

### 💫 Нейтрализация дефектов

Все перечисленные дефекты **нейтрализуются**, если:
- **Управитель асцендента** аспектирует лагну
- **Юпитер** (Гуру) аспектирует лагну

> **Важно:** Если лагну аспектирует либо её управитель, либо Юпитер, указанные дефекты не проявляются.
