# Дефекты брачной мухурты и предсвадебные обряды

## 🚫 Дефекты, которых следует избегать в благоприятных делах (Шлоки 107-109)

### Природные и астрономические дефекты:

| Категория | Конкретные дефекты |
| :--------- | :----------------- |
| **Природные аномалии** | Внезапное падение деревьев/домов, метеоритный дождь, сильные бури, молнии, гром из ясного неба, землетрясения, гало вокруг Луны/Солнца, крики шакалов |
| **Социальные беспорядки** | Волнения в деревнях, революционные инциденты |
| **Временные дефекты** | Дагдха-титхи, Ваатипат, вдовство |
| **Йогические дефекты** | Злые йоги, неблагоприятные сочетания |

### Астрологические дефекты:

| Тип дефекта | Описание |
| :---------- | :------- |
| **Планетные положения** | Заход Луны, Венеры, Юпитера; Дакшинаяна |
| **Титхи и накшатры** | Взлёт и падение титхи, накшатры; Ганданта лагны |
| **Санкранти** | День солнечного перехода |
| **Положения управителей** | Хозяин асцендента/навамши в 8-м доме; в 6-м доме |
| **Вредоносные планеты** | В лагне, хоре, дрешкане, навамше, двадашамше, триншамше |
| **Особые доши** | Паат, Кхарджур, Джамитра, Латта, Ведха, Бан, Упграха, Папакарттари |

### Дефекты накшатр:
- Накшатра с **жестокой планетой**
- Накшатра, **покинутая жестокой планетой**
- Накшатра, куда **вот-вот войдёт жестокая планета**
- Накшатра **солнечного/лунного затмения**
- Накшатра, где **поднялся Кету**
- Накшатра, начинающаяся во время **заката**
- 14-я накшатра от накшатры, где произошла **битва между планетами**

## 🏠 Въезд невесты в дом жениха (Шлока 111)

### Благоприятное время для въезда невесты:

| Период | Рекомендуемые дни | Примечания |
| :----- | :--------------- | :--------- |
| **В течение 16 дней после свадьбы** | Чётные дни: 2, 4, 6, 8, 10, 12, 14 | Считается благоприятным |
| **После 16 дней** | В нечётные дни: 5, 7, 9, 16 | Также благоприятно |
| **После акшайи** (особого года) | В любое желаемое время | В благоприятные дни |

### Особые указания:
- **Первые 16 дней:** предпочтительны чётные дни
- **После 16 дней:** можно использовать нечётные дни
- **После особого года:** возможен выбор времени по желанию
