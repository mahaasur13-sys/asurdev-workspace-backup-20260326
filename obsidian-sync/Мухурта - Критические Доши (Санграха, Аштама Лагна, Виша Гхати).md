# Мухурта - Критические Доши (Санграха, Аштама Лагна, Виша Гхати)

Эта заметка описывает несколько очень серьезных дефектов (дош) в карте брака (Виваха Мухурта), которые могут привести к бедности, смерти, отсутствию гармонии и другим несчастьям.

## 1. Санграха Доша (Соединение с Луной)

Эта доша возникает, когда в карте брака [[Чандра|Луна]] находится в соединении с другими планетами. Результат зависит от планеты.

| Планета в соединении с Луной | Результат для пары |
| :--- | :--- |
| *Солнце* | Бедность |
| *Марс* | Смерть |
| *Меркурий* | Благоприятно |
| *Юпитер* | Счастье |
| *Венера* | Появление "второй женщины" (соперницы) |
| *Сатурн* | Отсутствие любви и привязанности |
| *Два или более вредителя* | Смерть |

> *Мнение Нарады (дополнительно):*
> - *Меркурий:* Потеря детей.
> - *Юпитер:* Потеря удачи.
> - *Сатурн:* Отречение (Саньяса).
> - *Раху:* Постоянные ссоры.
> - *Кету:* Постоянные страдания и бедность.

*Общее правило:* Результат будет благоприятным, если Луна сильна (в экзальтации, своем знаке или знаке друга) и соединена с бенефиками. В противном случае — результат неблагоприятен.

## 2. Аштама Лагна Доша (Доша 8-й Лагны)

*Основное правило:* [[Лагна]] брака *не должна быть 8-м знаком* от натальной Лагны или от натального знака Луны ([[Джанма Раши]]) жениха или невесты.

### Отмена (Парихара) Аштама Лагна Доши

Эта серьезная доша считается отмененной, если выполняется *одно из* следующих условий:

1. *Единство или Дружба Управителей:*
   - Управитель натального знака (Лагны или Раши) и управитель 8-го от него знака — это *одна и та же планета* (например, для [[Тула|Весов]] 8-й знак — [[Вришабха|Телец]], управитель обоих — Венера).
   - Или управители этих двух знаков являются *взаимными друзьями* (например, для [[Симха|Льва]] 8-й знак — [[Мина|Рыбы]], управители Солнце и Юпитер — друзья).

2. *Особые Знаки-Исключения:*
   Если 8-й по счету знак (который становится Лагной брака) является одним из следующих, доша *не возникает*:
   - [[Мина]] (Рыбы)
   - [[Вришабха]] (Телец)
   - [[Карка]] (Рак)
   - [[Вришчика]] (Скорпион)
   - [[Макара]] (Козерог)
   - [[Канья]] (Дева)

> *Результат при отмене:* Если брак заключается при отмененной Аштама Лагна Доше, жена будет благословлена хорошими детьми, долголетием, прекрасным домом и счастьем.

## 3. Дополнительные Доши 8-го и 12-го Домов

Эти правила расширяют концепцию Аштама Лагны. Лагна брака *не должна быть*:

- *Связана с 8-м домом натала:*
    - Навамшей 8-го дома натальной карты.
    - Занята управителем 8-го дома натальной карты.
    - *Результат:* Не приносит блага.

- *Связана с 12-м домом натала:*
    - 12-м домом натальной карты.
    - Навамшей 12-го дома.
    - Занята управителем 12-го дома.
    - *Результат:* Приносит ссоры между супругами.

## 4. Виша Гхати Доша (Ядовитые Периоды Накшатр)

*Виша Гхати* — это определенный короткий промежуток времени внутри каждой Накшатры, который считается ядовитым и непригодным для любых начинаний. В тексте приведены следующие значения (1 Гхати ≈ 24 минуты):

| Накшатра | Ядовитый период (Гхати) |
| :--- | :--- |
| [[Ашлеша]], [[Джьештха]], [[Ревати]] | 30-я гхатика |
| [[Бхарани]] | 40-я гхатика |
| [[Сатабхиша]] | 32-я гхатика |
| [[Ашвини]] | 50-я гхатика |
| [[Анурадха]], [[Шатабхиша]] | 18-я гхатика |
| [[Криттика]], [[Пушья]], [[Читра]], [[Уттара Ашадха]], [[Пунарвасу]] | 20-я гхатика |
