# Мухурта для путешествий (Ятра) - продолжение

## 🌟 Благоприятные и неблагоприятные положения Солнца

### Влияние положения Солнца на путешествие:

| Положение Солнца | Результат путешествия |
| :--------------- | :------------------- |
| **Стрелец, Овен, Лев** | Благоприятное |
| **Козерог, Водолей, Близнецы, Дева, Телец, Весы** | Умеренное |
| **Рак, Скорпион, Рыбы** | Неблагоприятное (долгое возвращение) |

## 🔢 Расчёт по накшатрам

### Метод расчёта:
1. **Определить разницу** между накшатрой рождения и накшатрой дня путешествия
2. **Разделить на 9**
3. **Проанализировать остаток**

### Неблагоприятные остатки:
- **1, 3, 5, 7** - неблагоприятны для путешествия

## 📅 Запрещённые и благоприятные титхи

## 🧭 Запреты по направлениям и дням

### Критические сочетания:

| День | Накшатра | Направление | Запрет |
| :--- | :------- | :---------- | :----- |
| **Понедельник/Суббота** | Джьештха | Восток | Запрещено |
| **Четверг** | Пурвабхадрапада | Юг | Запрещено |
| **Воскресенье** | Рохини | Запад | Запрещено |
| **Пятница** | Уттара Пхалгуни | Север | Запрещено |
| **Среда** | Марс | Север | Запрещено |

> **Примечание:** Разумный человек, желающий богатства, победы и долгой жизни, должен избегать этих сочетаний.

## ⏰ Благоприятные и неблагоприятные периоды дня

### Деление дня и соответствующие накшатры:

| Период дня | Накшатры | Благоприятность |
| :--------- | :------- | :------------- |
| **Первая часть дня** | Уттара, Рохини, Вишакха, Криттика | Неблагоприятно для правителей |
| **Полдень** | Мула, Джьештха, Ардра, Ашлеша | Неблагоприятно |
| **Вторая половина дня** | - | Неблагоприятно |
| **Ранняя ночь** | Митра | Неблагоприятно |
| **Полночь** | - | Неблагоприятно |
| **Конец ночи** | - | Неблагоприятно |

### Всегда благоприятные накшатры:
