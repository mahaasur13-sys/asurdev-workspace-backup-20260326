# Мухурта для путешествий (Ятра)

## 👥 Применимость для всех категорий людей

**Путешествие необходимо всем** - от царей до простых людей, поскольку ни одна работа не может быть выполнена без путешествий.

### Два типа путешественников:
1. **Цари с известным временем рождения** - анализ натальной карты
2. **Люди с неизвестным временем рождения** - использование:
   - Прашна-астрологии (анализа момента вопроса)
   - Шакун (примет)
   - Раджа-йог в гороскопе

## ✅ Благоприятные йоги для путешествия

### Для царей с известным временем рождения:

| Условие | Результат |
| :------ | :-------- |
| **Знак рождения** в Прашна-лагне | Победа |
| **Управитель знака рождения** в Прашна-лагне | Победа |
| **Прашна-лагна** в 3,6,10,11 доме от знака рождения | Победа |

### Для всех путешественников (по Прашна-лагне):

| Благоприятный показатель | Результат |
| :---------------------- | :-------- |
| **Знак врага** в 4/7 доме от Прашна-лагны | Победа |
| **Управитель знака врага** в 4/7 доме | Победа |
| **3,6,10,11 дом от знака врага** в 4/7 доме | Победа |
| **Благотворные планеты** в Прашна-лагне | Победа |
| **Асцендент** в Близнецах, Льве, Деве, Весах, Скорпионе, Водолее | Победа |
| **Красивое окружение** во время вопроса (земля, цветы, храм) | Победа |
| **Благоприятные знамения** (звуки, видения) | Победа |
| **Вопрос задан с уважением** | Победа |
| **Асцендент** в Овне, Раке, Весах, Козероге с аспектами благотворных | Победа |

## ⚠️ Неблагоприятные йоги для путешествия

### Критические комбинации:

| Неблагоприятная конфигурация | Последствие |
| :--------------------------- | :---------- |
| **Луна или Марс** в Прашна-лагне с аспектом **Сатурна** | Поражение/Разрушение |
| **Солнце** в Прашна-лагне + **Луна** в 7/8 доме от него | Поражение/Разрушение |
| **Вредоносные планеты** в 4,7,8 домах от Прашна-лагны | Поражение/Разрушение |

## 🧭 Определение направления путешествия

### Метод 1: По силе планет
- Если **ни одна из планет** (Солнце, Луна, Марс, Юпитер, Венера) не находится в триконах (5,9 дома)
- **Выбирается направление** самой сильной планеты из перечисленных

### Метод 2: По положению в 5-м доме
- Если в **5-м доме** от Прашна-лагны находится сильная планета
- **Направление** определяется по этой планете

### Примечание:
Сила планеты определяется её положением в шадваргах (шести системах деления) и балой (силой).
