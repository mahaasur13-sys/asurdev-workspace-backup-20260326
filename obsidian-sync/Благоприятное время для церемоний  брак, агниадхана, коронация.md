# Благоприятное время для церемоний: брак, агниадхана, коронация

## ⚠️ Избегание дефекта Венеры (Шукра доша)

### Исключения для дефекта Венеры:

| Ситуация | Влияние дефекта Венеры |
| :------- | :-------------------- |
| **Беспорядки** в городе/деревне по пути | Не проявляется |
| **Паломничество** или **королевский скипетр** | Не проявляется |
| **Дом отца** для женщин с зрелой грудью и менструацией | Не проявляется |
| **Семьи мудрецов**: Бхригу, Ангира, Ватса, Васиштха, Кашьяпа, Атри, Бхарадваджа | Не проявляется |

> **Примечание:** В перечисленных случаях дефект Венеры (Шукра доша) не считается пагубным.

## 🔥 Церемония агниадханы (зажигание священного огня)

### Благоприятные условия:

**Накшатры:**
- Криттика
- Вишакха
- Рохини
- Все три Уттары
- Ревати
- Мригашира
- Джьештха
- Пушья

**Период:** Уттараяна (северный путь Солнца)

### Неблагоприятные условия:

**Титхи:**
- 4-я (Чатуртхи)
- 9-я (Навами)
- 14-я (Чатурдаши)

**Планетные положения:**
- Луна, Марс, Юпитер, Венера в:
  - Низших положениях
  - Побеждённых состояниях
  - Вражеских знаках

## ✅ Очищение асцендента для агниадханы

### Запрещённые асценденты:
- **Рак** (Карката)
- **Козерог** (Макара)
- **Рыбы** (Ми́на)
- **Водолей** (Кумбха)

### Благоприятные планетные положения:

| Планеты | Благоприятные дома |
| :------ | :----------------- |
| **Солнце** | 5, 9, 1 |
| **Луна, Юпитер, Марс** | 4, 7, 10 |
| **Меркурий, Венера, Сатурн, Раху, Кету** | 3, 11, 6, 10 |

> **Исключение:** 8-й дом от асцендента рождения

## 👑 Йоги для совершающего яджну (Шлока 3)

### Благоприятные конфигурации:

| Планета | Положение | Результат |
| :------ | :-------- | :-------- |
| **Юпитер** | В денежном знаке или Овне | Совершит яджну |
| **Марс** | В асценденте, 7-м или 10-м доме | Совершит яджну |
| **Луна** | В 3-м, 6-м или 11-м доме | Совершит яджну |
| **Солнце** | В 3-м, 6-м или 11-м доме | Совершит яджну |

## 🏛️ Коронация (Раджабхишека)

### Благоприятные условия:
- **Период:** Уттараяна (северный путь Солнца)
- **Планеты:** Юпитер, Венера - сильные и восходящие
- **Асцендент:** Сильный, с благотворными планетами
- **Время:** День (не ночь)
