# Минимальные технические требования для ПК-приложения планировщиков

**Платформа:** Кроссплатформенное desktop-приложение (Windows/Linux/macOS)

**Минимальные требования:**

**ОС:**

- Windows 7 (x86/x64)
    
- Ubuntu 18.04 LTS или новее
    
- macOS 10.14 Mojave
    

**Аппаратные требования:**

- Процессор: 2-ядерный CPU 1.5 GHz
    
- Оперативная память: 4 ГБ RAM
    
- Место на диске: 2 ГБ (с учетом зависимостей)
    

**Зависимости:**

- .NET Framework 4.7.1 (для Windows)
    
- Python 3.6+ (в комплекте)
    
- Java Runtime 11+ (опционально для отдельных планировщиков)
    

**Дополнительно:**

- Интернет-соединение (для установки/обновлений)
    
- Разрешение экрана: 1280x720
