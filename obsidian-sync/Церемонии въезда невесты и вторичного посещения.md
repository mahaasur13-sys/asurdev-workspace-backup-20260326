# Церемонии въезда невесты и вторичного посещения

## 🏠 Въезд невесты в дом жениха (Вадхуправеша)

### Благоприятные накшатры для въезда:
**Перечень благоприятных созвездий:**
- Рохини
- Все три Уттары (Уттара Пхалгуни, Уттара Ашадха, Уттара Бхадрапада)
- Ашвини
- Пушья
- Хаста
- Читра
- Анурадха
- Мригашира
- Шравана
- Дхаништха
- Мул
- Магха
- Свати

### Благоприятные титхи:
- **4-я** (Чатуртхи)
- **9-я** (Навами)
- **14-я** (Чатурдаши)

### Неблагоприятные дни:
- Воскресенье
- Вторник
- **Меркурий** (по мнению некоторых ачарьев) - неблагоприятен даже днём

## ⚠️ Неблагоприятные месяцы для пребывания

### Последствия пребывания в определённые месяцы:

| Месяц | Место пребывания | Последствие |
| :---- | :--------------- | :---------- |
| **Джьештха** | Дом мужа | Губит старшего брата мужа |
| **Малмаса** (доп. месяц) | Дом мужа | Губит мужа |
| **Ашадха** | Дом мужа | Губит свекровь |
| **Пауша** | Дом мужа | Губит свекра |
| **Кшая** | Дом мужа | Губит своё тело |
| **Чайтра** | Дом отца | Губит отца |

### Рекомендации:
- **Первые месяцы** после свадьбы следует проводить в доме отца в: Джьештха, Малмаса, Ашадха, Пауша, Кшая
- **В доме мужа** - в первый месяц Чайтра
- **Исключение:** Если потенциально пострадавшие лица не живы, пребывание не несёт греха

## 🔁 Вторичный въезд невесты (Двирагамана)

### Благоприятные условия:

| Параметр | Благоприятные значения |
| :------- | :--------------------- |
| **Годы** | Нечётные (3-й, 5-й и т.д.) |
| **Солнце** | Кумбха, Скорпион, Овен |
| **Планеты** | Солнце и Юпитер чистые |
| **Дни недели** | Понедельник, среда, четверг, пятница |
| **Асцендент** | Близнецы, Рыбы, Дева, Весы, Телец |

### Благоприятные накшатры:
Ашвини, Пушья, Хаста, Рохини, все три Уттары, Шравана, Дхаништха, Шатабхиша, Пунарвасу, Свати, Мул, Читра, Анурадха, Мригашира, Ревати

## ⛔ Дефект положения Венеры

### Критическое положение:
- **Венера спереди** (восток)
- **Венера справа** (юг)

### Последствия для разных категорий женщин:

| Категория женщины | Последствие |
| :---------------- | :---------- |
| **Беременная** | Смерть ребёнка |
| **С детьми** | Смерть ребёнка |
| **Новая невеста** | Бесплодие |
| **Беременная** | Выкидыш, разрушение матки |

### Запрет:
Женщинам не следует совершать вторичный въезд в дом мужа при таком положении Венеры.
