# Расчёт благоприятного времени для брака (Ишта-кала)

## ⏰ Метод расчёта Ишта-калы (Шлока 105-106)

### Основной принцип:
Когда **асцендент** и **Солнце** находятся в **одном знаке**:

1. **Вычитаем** положение Солнца из положения асцендента
2. **Умножаем** разницу на время восхода для данного знака
3. **Делим** на 30

**Формула:**          
### Особые случаи:

#### Случай 1: Асцендент опережает Солнце
- Результат показывает **время от восхода Солнца** до благоприятного момента

#### Случай 2: Асцендент отстаёт от Солнца
- Результат показывает **остаток ночи** до восхода
- **Ишта-кала = 60 - полученное значение**

### Примеры расчётов:

#### Пример 1 (Лагна > Сурья):
- **Положение Лагны:** 9°25'6.36"
- **Положение Сурьи:** 9°16'59.26"
- **Разница:** 0°8'7.10"
- **Время восхода Козерога:** 303 пал
- **Расчёт:** (0°8'7.10" × 303) ÷ 30 = 82 пал 22 випал
- **Результат:** Благоприятное время через 82 пал 22 випал после восхода

#### Пример 2 (Лагна < Сурья):
- **Положение Лагны:** 9°22'45.36"
- **Положение Сурьи:** 9°26'50.40"
- **Разница:** 0°4'5.4"
- **Расчёт:** (0°4'5.4" × 303) ÷ 30 = 41 пал 15 випал
- **Итог:** 60 - 41.15 = 59 пал 18 випал 44 випалы
- **Результат:** Благоприятное время за 59 пал до восхода

## 🚫 Дефекты, которых следует избегать (Шлока 107)

### Критические дефекты для благоприятных дел:

| Категория дефектов | Конкретные примеры |
| :----------------- | :----------------- |
| **Временные дефекты** | Утпат, Самха, Пата, Дагдха-титхи |
| **Йогические дефекты** | Неблагоприятные йоги, Шанта-йога |
| **Лунные дефекты** | Неблагоприятные положения Луны |
| **Солнечные дефекты** | Неблагоприятные положения Солнца |
| **Дефекты титхи** | Кшая-титхи (разрушительные титхи) |
| **Навамша дефекты** | Ганда-анта, Савишти, Санкранти |
| **Планетные дефекты** | Вражеские планеты в навамше |

### Дополнительные неблагоприятные факторы:
- **Кхара-джама** (жесткие аспекты)
- **Даша-йога** (неблагоприятные периоды)
- **Джамитра-доша** (дефекты 7-го дома)
