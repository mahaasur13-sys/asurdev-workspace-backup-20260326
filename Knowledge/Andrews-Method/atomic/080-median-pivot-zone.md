# 080: Разворотная Зона Срединной Линии (Median Line Pivot Zone)

## Конспект

**Суть:** Инструмент для оценки **времени** достижения ценой Срединной линии.

> *"Как долго трейдер должен ждать, пока цена дойдет до Медианы?"*

## Построение

```
1. Провести Срединную линию (A, B, C)
2. Посчитать количество баров: A→B и B→C
3. Взять БОЛЬШЕЕ значение
4. От точки C отложить это количество баров вперёд по медиане
5. Построить прямоугольник от C до этой точки
```

```
    |←── 15 bars (max) ──→|
    C───────────────────[POINT]
    │                      ↑
    │                     矩形 = Разворотная Зона
    ↓
```

## Интерпретация

| Условие | Значение |
|---------|----------|
| Цена коснулась медианы **внутри зоны** | Рынок в цикличном режиме — можно торговать |
| Цена **не достигла** медианы | Рынок не в фазе — осторожность |
| **Множественные касания** | Не входить на первом касании |

## Правила

- ⚠️ Касание **не гарантирует** разворот
- ✅ Касание **подтверждает** цикличный режим
- ⏱️ Зона = временной ориентир, а не точное время
