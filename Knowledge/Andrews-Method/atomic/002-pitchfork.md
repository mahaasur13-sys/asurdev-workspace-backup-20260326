# 002: Вилы Эндрюса (Andrews Pitchfork)

## Конспект

**Суть:** Вилы — это канал, образованный Срединной линией и двумя параллельными линиями, проведёнными через точки B и C.

## Построение

```
1. Провести Срединную линию от A через середину BC
2. Через точку B провести линию, параллельную медиане
3. Через точку C провести линию, параллельную медиане
```

```
        B--------|------- B (параллель)
       /|        |        /
      / |        |       / |
     /  |        |      /  |
    /   |        |     /   |
   A----|--- C --|----A    C
        ↑        медиана
        Срединная линия
```

## Компоненты

| Компонент | Описание |
|-----------|----------|
| **Срединная линия** | Центральная линия (магнит цены) |
| **Верхняя параллель** | Линия через B |
| **Нижняя параллель** | Линия через C |
| **Ширина вил** | Расстояние между B и C |

## Сигнальные линии (Trigger Lines)

Дополнительные линии внутри вил:
- **Верхняя сигнальная** — 1/2 ширины сверху
- **Нижняя сигнальная** — 1/2 ширины снизу
