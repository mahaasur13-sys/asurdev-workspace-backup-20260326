# 095: Супер-Вилы (Super Pitchforks)

## Конспект

**Суть:** Объединение **Вил Эндрюса** и **Метода Действия-Реакции** для получения более надёжных сигналов на пересечении линий.

> *"Линии Реакции, проведенные по последним данным, работали лучше всего"*

## Построение

```
1. Строим стандартные Вилы по точкам A, B, C
2. Линия B-C используется как Линия Центра для A-R
3. Расстояние от A до линии B-C = шаг для линий реакции
4. Проводим линии реакции (RL1, RL2, RL3...) параллельно линии B-C
```

## Ключевая точка D

**Пересечение Срединной линии + Линии реакции 1 = зона высокого потенциала**

```
         RL2
        /
    RL1/------- D (пересечение медианы и RL1)
      /
     /
A---/--- B
    \
     \----- C
         \
          RL1
```

## Сигналы Супервил

| Сигнал | Описание |
|--------|----------|
| **Точка D** | Пересечение медианы и RL1 — искать разворот |
| **RL + Warning Line** | Пересечение линии реакции и предупреждающей линии |
| **Шип на медиане** | Резкое касание и отскок |
