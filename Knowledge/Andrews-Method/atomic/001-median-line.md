# 001: Срединная линия (Median Line)

## Конспект

**Суть:** Срединная линия — это линия, проведённая от разворотной точки **A** через середину отрезка **B-C**, где B и C — экстремумы колебания.

**Ключевое свойство:** Эндрюс утверждал, что цена достигает Срединной линии в **~80% случаев**.

## Построение

```
1. Найти три разворотные точки: A (начало), B (максимум/минимум), C (противоположный экстремум)
2. Провести линию от A через середину BC
3. Это и есть Срединная линия
```

```
        B
       /|\
      / | \
     /  |  \
    /   |   \
   /    |    \
  A-----|-----C
        ↑
   Срединная линия
```

## Физический смысл

- Срединная линия представляет "линию равновесия" колебания
- Цена стремится вернуться к равновесию (как маятник)
- Это **магнит цены**

## Правила торговли

| Правило | Условие | Действие |
|---------|---------|----------|
| 1 | Цена идёт к медиане | Ждать признаков разворота |
| 2 | Цена не дошла и развернулась | Цель = предыдущий экстремум |
| 3 | Цена пробила медиану | Ожидать движения к противоположной параллели |
