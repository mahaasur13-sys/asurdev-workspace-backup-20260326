# Ключевые концепции Майкла Манкаси

## 1. Планетарные кузнецы (Planetary Forgers)

Главная инновация Манкаси. Кузнец — это планета, которая управляет управителем знака на куспиде дома.

```
Пример: 
- Солнце в IX доме во Льве
- Управитель IX дома = Солнце
- Кузнец Солнца? Смотрим Лев → управитель = Солнце
- Солнечный кузнец сам по себе
```

### Система силы:
1. **Прямой** — планета в своём знаке → максимальная сила
2. **В обители** — в знаке своей обители
3. **В экзальтации** — высшая степень силы
4. **В изгнании** — ослаблена
5. **В падении** — минимальная сила

## 2. Перехваченные знаки

Когда знак находится между двумя домами, он "перехвачен" и его энергия подавлена.

**Признаки перехвата:**
- Знак не содержит куспид дома
- Оба угла дома в соседних знаках

**Влияние перехвата:**
- Сложность проявления качеств знака
- Ограничение в сфере дома
- Компенсация через диспозиторов

## 3. Дом как тематическая сфера

Манкаси подчёркивал: **дом показывает конкретную сферу жизни**, а не абстрактную идею.

| Дом | Ключевая тема |

| I | Кто я есть |
| II | Мои ресурсы |
| III | Моё окружение |
| IV | Мой корень |

| VI | Моё служение |

| IX | Мои поиски |

| XII | Мои ограничения |

## 4. Система оценки домов

### Сильные дома:
- Управитель в угловом положении
- Много планет в доме
- Управитель в обители/экзальтации

### Слабые дома:
- Управитель в падении/изгнании
- Перехваченный знак
- Аспекты к малефикам
- Мало планет
