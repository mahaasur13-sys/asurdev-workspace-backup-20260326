# Gann Theory (Теория Ганна)

## Определение

**Gann Theory** — комплексный метод анализа финансовых рынков, созданный Уильямом Делбертом Ганном (W.D. Gann, 1878–1955), объединяющий геометрию, астрологию, древнюю математику и анализ времени-цены. Ганн утверждал, что рынки движутся в соответствии с геометрическими паттернами и временными циклами, которые можно предсказать с математической точностью.

> *"When time and price square, a change in trend is imminent."* — W.D. Gann

## Ключевые инструменты

### 1. Gann Angles (Углы Ганна)

Девять углов, каждый из которых представляет определённое соотношение цены и времени:

| Угол | Соотношение | Градусы | Значение |
|------|-----------|---------|---------|
| **1×8** | 1 единица цены : 8 единиц времени | 7.125° | Очень медленный восходящий тренд |
| **1×4** | 1:4 | 14.036° | Медленный восходящий тренд |
| **1×3** | 1:3 | 18.435° | Умеренно-медленный |
| **1×2** | 1:2 | 26.565° | Умеренный |
| **1×1** | 1:1 | **45°** | **Master Angle** — баланс цены и времени |
| **2×1** | 2:1 | 63.435° | Сильный восходящий тренд |
| **3×1** | 3:1 | 71.565° | Очень сильный |
| **4×1** | 4:1 | 75.964° | Экстремальный |
| **8×1** | 8:1 | 82.875° | Параболический (неустойчивый) |

> **Мастер-угол 1×1 = 45°** означает, что одна единица цены равна одной единице времени. Это линия идеального баланса. Цена выше 1×1 — бычий рынок; ниже — медвежий.

### 2. Square of Nine (Квадрат Девяти)

Геометрическая спиральная структура для определения **уровней поддержки/сопротивления** и **временных целей**.

```
Структура (центр → край):
     37  38  39  40  41
     36  17  18  19  42
     35  16   1   2  43
     34  15   4   3  44
     33  32  31  30  45
```

**Формула определения уровней:**

```
n² + n + 1  = значения на диагоналях
```

где n — шаг от центра.

**Применение:**
- Добавить текущую цену к числу в Квадрате → следующий resistance
- Вычесть → следующий support
- Деление на 2, 4, 8 → промежуточные уровни

### 3. Sacred Geometry (Священная геометрия)

#### Гексаграмма

Пересечение двух треугольников (вверх + вниз) — древний символ единства противоположностей. Используется для определения **точек баланса** между бычьим и медвежьим давлением.

```
        ▲ (стихия огня / дух)
       / \
      /   \
     /     \
    ▼ (стихия воды / материя)
```

#### Cardinal Cross (Кардинальный крест)

Деление квадрата на 4 равных квадранта:

| Точка | % от диапазона | Значение |
|-------|---------------|---------|
| 0% | 0° | Начало |
| 25% | 90° | Первая четверть |
| 50% | 180° | Экватор (сопротивление) |
| 75% | 270° | Третья четверть |
| 100% | 360° | Полный цикл |

#### Golden Ratio (Золотое сечение)

```
φ = (1 + √5) / 2 ≈ 1.6180339887
φ⁻¹ = 1/φ ≈ 0.6180339887
```

Внутренние треугольники гексаграммы масштабируются на **φ⁻¹**. Ключевые точки: **61.8%** и **38.2%** от диапазона.

### 4. 8-частное деление цикла

Каждый цикл делится на 8 равных частей — каждая часть имеет своё значение:

| Часть | % цикла | Градус | Значение |
|-------|---------|--------|---------|
| 1/8 | 12.5% | 45° | Начало (рождение) |
| 2/8 | 25% | 90° | Первое сопротивление |
| 3/8 | 37.5% | 135° | Откат |
| 4/8 | 50% | 180° | Экватор — критическая точка |
| 5/8 | 62.5% | 225° | Второе сопротивление |
| 6/8 | 75% | 270° | Финальный откат |
| 7/8 | 87.5% | 315° | Предразворот |
| 8/8 | 100% | 360° | Завершение (смерть/новый цикл) |

## Применение в трейдинге

### Определение тренда

```
Цена над 1×1 (45°) → бычий рынок
Цена под 1×1 (45°) → медвежий рынок
Угол наклона > 45° → сильный бычий
Угол наклона < 45° → слабый бычий
```

### Уровни поддержки/сопротивления

1. Определить текущую цену
2. Найти её в Квадрате Девяти
3. Соседние числа = ближайшие уровни
4. Числа на диагоналях = ключевые уровни

### Временные цели

- Когда цена и время " квадратируются" (цена × время = квадрат) → вероятен разворот
- Циклы: 7, 10, 14, 20, 28, 30, 45, 60, 90, 120, 180 дней — ключевые временные окна
