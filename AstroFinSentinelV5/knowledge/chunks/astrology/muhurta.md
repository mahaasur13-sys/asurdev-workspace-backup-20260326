# Muhurta — Искусство выбора времени

## Overview

Muhurta (मुहूर्त) — ведическая наука о выборе благоприятного момента для любого начинания. Трейдинг — это бизнес, и выбор правильного времени для входа критически важен.

## Связь с Choghadiya и Nakshatra

Muhurta Score = Choghadiya Score + Nakshatra Score + Yoga Score

| Компонент | Вес | Описание |
|-----------|-----|---------|
| Choghadiya | 50% | Период дня |
| Nakshatra | 30% | Лунная стоянка |
| Yoga | 20% | Комбинация планет |

## Yoga (не путать с физическими упражнениями)

В астрологии Yoga — это комбинация положений планет. 
Существует 27 Yoga, каждая с своим влиянием.

### Благоприятные Yoga для трейдинга:
- **Shubha** — благоприятная
- **Amrita** — нектарная
- **Siddhi** — успешная
- **Dhruva** — устойчивая

### Неблагоприятные Yoga:
- **Vyaghata** — препятствия
- **Harshana** — разрушительная
- **K Duval** — несёт потери

## Формула Muhurta Score

```
Muhurta Score = 5.0 (base)
              + Choghadiya_modifier  # -3 to +3
              + Nakshatra_modifier   # -2 to +2
              + Yoga_modifier        # -1 to +1
```

| Muhurta Score | Рекомендация |
|---------------|--------------|
| 8.0 — 10.0 | Отличное время для входа |
| 6.0 — 7.9 | Хорошее время |
| 4.0 — 5.9 | Нейтральное |
| 2.0 — 3.9 | Плохое время |
| 0.0 — 1.9 | Не входить |

## Trade Entry Rules

### ✅ ENTER (Score >= 6.5):
- Choghadiya: Amrita, Shubha, Labha
- Nakshatra: Ashwini, Rohini, Pushya, Hasta, Uttara Ashadha, Revati
- Good Yoga

### ⚠️ WAIT (Score 4.0-6.4):
- Choghadiya: Chandra
- Neutral Nakshatras
- Mixed Yoga

### ❌ AVOID (Score < 4.0):
- Choghadiya: Marana, Vyatipata, Kaala
- Bad Nakshatras: Bharani, Ardra, Ashlesha, Jyeshtha
- Bad Yoga

## Пример расчёта

```
Дата: 2026-03-24 14:00 MSK
Choghadiya: Amrita (+3.0)
Nakshatra: Rohini (excellent, +2.0)
Yoga: Shubha (+1.0)

Muhurta Score = 5.0 + 3.0 + 2.0 + 1.0 = 11.0 → capped at 10.0
Рекомендация: ✅ EXCELLENT TIME TO ENTER
```

## Важные запреты

1. **Не входить в Marana Choghadiya** независимо от технического анализа
2. **Не входить в Nakshatra Ashlesha** — ловушки и обман
3. **Не входить в Vyatipata** — опасные развороты
4. **Избегать Rahukaal** для новых позиций
