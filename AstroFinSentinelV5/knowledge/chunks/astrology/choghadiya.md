# Choghadiya — Ведические периоды дня

## Overview

Choghadiya (चौघडिया) — это система деления дня на 8 периодов, каждый из которых имеет свою энергетику. Используется для определения благоприятного времени для различных начинаний.

## 8 Типов Choghadiya

| Тип | Качество | Описание | Торговля |
|-----|---------|----------|----------|
| **Amrita** | ⭐ Лучший | Нектарное время — наиболее благоприятно | ✅ Отлично |
| **Shubha** | 🟢 Хороший | Благоприятное время | ✅ Хорошо |
| **Labha** | 🟢 Хороший | Время прибыли | ✅ Хорошо |
| **Chandra** | 🟢 Хороший | Время Луны — хорошо для эмоциональных дел | ✅ Приемлемо |
| **Kaala** | 🔴 Плохой | Чёрное время | ❌ Избегать |
| **Marana** | ⛔ Худший | Время смерти — категорически избегать | ❌ Нет |
| **Vyatipata** | 🔴 Опасный | Опасный период | ❌ Нет |
| **Parivesha** | 🔴 Проблемный | Время трудностей | ❌ Нет |

## Дневные vs Ночные

- **Дневные** (от восхода до заката): 8 периодов
- **Ночные** (от заката до восхода): 8 периодов
- Каждый период ≈ 90 минут (зависит от широты и времени года)

## Правила для трейдинга

### ✅ Благоприятно (ENTER):
- Amrita Choghadiya — входить в сделки
- Shubha — хорошие входы
- Labha — особенно хорошо для profit-taking

### ❌ Неблагоприятно (AVOID):
- Marana — НИКОГДА не входить
- Vyatipata — избегать новых позиций
- Kaala — только закрытие позиций

## Muhurta Score

Muhurta Score = Choghadiya Score + Nakshatra Score

| Choghadiya | Score |
|------------|-------|
| Amrita | +3.0 |
| Shubha | +2.0 |
| Labha | +1.5 |
| Chandra | +1.0 |
| Kaala | -2.0 |
| Marana | -3.0 |
| Vyatipata | -2.5 |
| Parivesha | -2.0 |

## Rahukaal (Рahu Kaal)

Отдельный неблагоприятный период, не связанный с Choghadiya:
- Каждый день ~1.5 часа
- День делится на 8 частей по 90 минут
- Rahukaal — последняя часть каждого дня

Rahukaal не является Choghadiya, но同样 важно учитывать для трейдинга.
