# Nakshatras — 27 Лунных Мансионов

## Overview

Nakshatras (नक्षत्र) — 27 лунных созвездий, через которые проходит Луна за ~27 дней. Каждое имеет уникальную энергетику, влияющую на начинания.

## Правила для Muhurta

### Лучшие Nakshatras для трейдинга:
1. **Ashwini** — быстрые результаты
2. **Rohini** — устойчивый рост
3. **Pushya** — стабильность
4. **Hasta** — творческие стратегии
5. **Uttara Ashadha** — победа
6. **Revati** — процветание

### Худшие Nakshatras для трейдинга:
1. **Bharani** — потери
2. **Ardra** — волатильность и убытки
3. **Ashlesha** — обман, ловушки
4. **Jyeshtha** — низкая точка

## Pada (Четверти)

Каждая Nakshatra делится на 4 Pada (части) по ~3°20'. 
Pada 1 и 3 обычно более благоприятны для начинаний.
