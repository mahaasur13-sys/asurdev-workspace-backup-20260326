# Elliott Wave Theory

## Core Principle

Markets move in 5-wave impulse patterns followed by 3-wave corrective patterns.

### Impulse Waves (5 waves)
- Wave 1: Initial move up
- Wave 2: Retraces Wave 1 (38-78%)
- Wave 3: Strongest wave, never shortest
- Wave 4: Retraces Wave 3 (38-78%), does not overlap Wave 1
- Wave 5: Final push

### Corrective Waves (3 waves)
- Wave A: First decline
- Wave B: Counter-trend rally
- Wave C: Final decline

## Fibonacci Ratios

| Wave | Target |
|------|--------|
| Wave 2 | 38.2%, 50%, 61.8%, 78.6% of Wave 1 |
| Wave 3 | 161.8%, 261.8% of Wave 1 |
| Wave 4 | 23.6%, 38.2% of Wave 3 |
| Wave 5 | 61.8%, 100%, 161.8% of Wave 4 |

## Trading Application
- Enter on Wave 2 or Wave 4 pullbacks
- Stop below Wave 1 start (for longs)
- Target Wave 3 extension or Wave 5
