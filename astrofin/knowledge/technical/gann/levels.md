# Gann Levels — Уровни Ганна

## Основы
Уровни поддержки/сопротивления на основе углов Ганна.

## Ключевые углы:
- 1x1 (45°) — главный тренд
- 2x1 (63.75°) — сильный рост
- 1x2 (26.25°) — коррекция
- 3x1 (75°) — экстремальный рост
- 4x1 (78.75°) — перекупленность

## Расчёт:
```
Support = High - (High - Low) * k
где k = 0.125, 0.25, 0.375, 0.5, 0.625, 0.75, 0.875
```
