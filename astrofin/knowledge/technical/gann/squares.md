# Gann Square of Nine

## Основы
Циклическая структура квадрата 9 Ганна для определения целей и разворотов.

## Расчёт уровней:
```
Level = SQRT(базовая_цена) + (шаг * N)
где N = 1, 2, 3, ... и SQRT(базовая_цена) ± шаг
```

## Применение:
- Цели = sqrt(price) * sqrt(price) ± step
- Время = углы квадрата (90°, 180°, 270°)
- Cycle = 360 дней (1 полный оборот)
