# Aspects — Аспекты

## Орбы (допуски):
| Аспект | Градусы | Орб |
|--------|---------|-----|
| Conjunction | 0° | 8-10° |
| Sextile | 60° | 4-6° |
| Square | 90° | 6-8° |
| Trine | 120° | 6-8° |
| Opposition | 180° | 8-10° |

## Значения:
- **Conjunction** ☌ — объединение энергий
- **Sextile** ⚹ — возможности
- **Square** ☐ — напряжение, активность
- **Trine** Δ — гармония, успех
- **Opposition** ☍ — конфликт, крайности

## Применение:
- Trine + Trine = благоприятное время
- Square + Square = повышенная волатильность
- Aspect к луне = эмоциональный фон
