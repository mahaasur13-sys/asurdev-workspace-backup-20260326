## Muhurta Selection по Nakshatra

### Ideal Nakshatras для входа
1. **Ashwini** — начинания, быстрые сделки
2. **Bharani** — инвестиции, накопление
3. **Rohini** — удержание позиции, рост
4. **Mrigashirsha** — поиск возможностей
5. **Pushya** — стабильные вложения
6. **Dhanishtha** — объёмы, крупные сделки
7. **Shatabhisha** — конфиденциальные операции

### Nakshatras для выхода
- **Uttara Shadha** — фиксация прибыли
- **Shravana** — завершение цикла
- **Mula** — стоп-лоссы, экстренный выход
